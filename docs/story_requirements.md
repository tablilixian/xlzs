# 激情故事写作规范

## 文件格式
- 每版存为 `docs/arousal_stories_v{number}.md`
- 首行 `# Arousal Stories v{number} - {主题}`
- 每条故事以 `## {标题}` 开头，标题独占一行，正文紧随

## 角色
- 男：阿贤（统一名字，被绿/被操控方）
- 女：甜甜（最常用）/ 小芳 / 婷婷 / 珍珍 / 丽丽 / 王姐 等，视场景按需搭配

## 叙事视角
- **优先女性第一人称**（"我"、"我们姐妹"、"老娘"等），称呼对方"阿贤"
- 女主口吻：极度强势、满嘴脏话、命令式、羞辱+掌控感
- 也可用男性第一人称（阿贤视角听/看），但仍以女性主动叙述为主

## 内容要求
- 两个方向：
  1. **绿帽/淫妻**：女主主动出轨/约炮/被干，阿贤旁观或事后知道
  2. **女强势/女主导**：女主碾压操控阿贤，命令他口交/自慰/跪着看，脏话羞辱
- 极度露骨：阴茎、阴道、操、鸡巴、骚逼等词直接使用
- 具体场景画面，对话+动作穿插，不能光叙述
- **每条 2500+ 字**（中文）
- 适合 TTS 朗读：句子节奏自然，标点清晰

## TTS 转换
- 引擎：Edge TTS（`zh-CN-XiaoxiaoNeural`），本地 Docker `travisvn/openai-edge-tts:latest` 端口 5050
- 分块：250 字以内，句号/问号/感叹号处切分
- 合并格式：**opus 24kbps / 24kHz mono**（约 1.3 MB/篇，10 min 以内）
- 输出目录：`sounds/voices/arousal_stories/`
- 命名规则：`story_v{version}_{index:02d}.opus`
