# 毕业旅行 · 第三天深夜 · 酒吧

民宿一楼的小酒吧。说是酒吧其实就是一个吧台加几组沙发——老板自己调的酒，没有什么酒单，你想喝什么告诉他就行。角落有一台老旧的投币点唱机——投一块钱可以选五首歌。灯光是暗红色的——几盏罩了红纱的壁灯把整个空间泡在暧昧的颜色里。窗户开着半扇——夜风把窗帘吹起来又落下去。

十个人在晚餐后散了一小时步——绕着民宿走了两圈消食——然后不约而同地聚到了酒吧里。没有人说"我们去喝一杯"——但每个人洗完澡换了干衣服之后都朝那个方向走。阿强最先到——他穿了一件白色的T恤和宽松的短裤坐在吧台的高脚凳上。老板正在擦杯子——他问阿强喝什么。阿强说随便——老板给他调了一杯金汤力。

其他人陆陆续续进来。佳佳穿着浴袍就来了——他说反正又不出门。嘉嘉穿了一件他的T恤——太大了领口滑到肩膀以下。静静穿了一件深色的长裙——很少看她穿裙子。阿敏多看了她两眼——她感觉到了但没有回应。

珍珍最后一个下来。她换了一条黑色的吊带裙——白天扎起来的头发散开了披在肩膀上。她走进来的时候所有人都安静了大约一秒——不是因为她穿得好看——是因为她脖子上那块红痕从锁骨延伸到领口的边缘，在暗红色的灯光下像一枚印章。

她径直走到吧台坐在阿强旁边——给我也来一杯跟他一样的。老板又做了一杯金汤力推到她面前。她喝了一口然后说——谁提议的泡吧。

佳佳——没有人提议。大家自己来的。

珍珍——那我们得做点什么。光喝酒没意思。

佳佳——你又有什么馊主意。

珍珍看了他一眼然后目光移到了沙发区的矮桌上。矮桌不大——大概六十公分见方——桌面是磨砂玻璃的，下面垂着一块深色的桌布。桌布垂到地面——完全遮住了桌面以下的空间。珍珍看着那块桌布大概过了三秒然后她说——我有一个游戏。

阿伟——你又来。

珍珍——很简单。每个人轮流当"做东"。做东的人选一个人——做一件不能让其他人看到的事。其他人正常聊天喝酒。如果做东的人被人发现他在做什么——他喝三杯。

佳佳——这什么破规则。什么叫"发现"？

珍珍——你猜。你猜对了就算。

阿强——猜对了怎么算。

珍珍——猜对了——做东喝三杯。猜错了——猜的人喝三杯。

阿敏——做东的人选的人需要配合吗。

珍珍——不需要。被选的人可以不配合。

阿敏——那他不配合怎么办。

珍珍——他会配合的。

游戏开始了。第一轮做东是珍珍。

她坐在吧台前面。阿强坐在她旁边。其他人散坐在沙发区和吧台之间——佳佳和嘉嘉在点唱机旁边选歌，静静和阿敏坐在靠窗的沙发上聊天，小芳和阿贤在另一组沙发上——小芳盘腿坐着阿贤的手放在她膝盖上，甜甜和阿伟在吧台的另一侧——甜甜正在跟老板聊天问他这款金汤力用的什么金酒。

珍珍的表情很放松——她在跟老板聊天问他是哪里人为什么来草原开民宿。她的左手放在吧台上右手放在自己腿上——看起来很正常。

但她的右手不在自己腿上。

在桌布下面——她的右手伸到了阿强的裤子上。她的手指在他大腿内侧慢慢游走——从膝盖一直滑到根部然后停在那个位置不动了。阿强端着酒杯在喝——他的表情也没有变化。他放下酒杯的时候杯底在吧台上放得重了一点点。

佳佳从点唱机那边回头看了一眼——他看到了什么呢。他什么都没看到。珍珍和阿强隔着至少一个凳子的距离坐着。她的表情很正常。但佳佳注意到了阿强放杯子的那一下——太重了。他笑了笑没有说。

珍珍的手隔着阿强的短裤握住了他已经硬起来的鸡巴的根部——拇指沿着茎身的轮廓来回滑动。阿强端起酒杯又喝了一口——他的喉结动了一下。

甜甜的声音从吧台另一侧传过来——她说——珍珍你的酒快没了要不要再来一杯。珍珍转过头回答她的时候她的手没有停——她的手指拉开了阿强短裤的松紧带——指尖碰到了他龟头。阿强的呼吸在那一瞬间改了拍子——很短——只有半秒——然后就恢复了。

但甜甜看到了那半秒。

甜甜端起酒杯走到珍珍旁边坐下——她说——我陪你喝。她坐下的时候低头看了一眼——桌布的边缘刚好垂到地面什么都看不到。她抬起头看着珍珍——珍珍冲她笑了一下。

甜甜说——做东的人不能光聊天吧。你选人了吗。

珍珍——选了。

甜甜——谁。

珍珍——你猜。

甜甜——阿强。

珍珍——猜错了。我选的是佳佳。

全场安静了一拍。佳佳从点唱机那边转过头——你选我干什么。

珍珍的右手从阿强的裤子里抽了出来。她端起酒杯喝了一口——我已经做完了。

佳佳——你做了啥。

珍珍——你要是猜到了你就赢了。

佳佳想了几秒——他的表情从不耐烦变成了有所谓的笑容——他说——你刚才摸的不是阿强。

珍珍——你猜对了还是猜错了你倒是说啊。

佳佳——我猜你摸的是阿贤。

珍珍——错了。喝三杯。

佳佳——那你到底摸的谁。

珍珍站起来——她走到吧台另一端在嘉嘉旁边坐下。她的嘴唇贴着嘉嘉的耳朵说了一句话——嘉嘉的表情没有变化——但她的耳朵红了。

没有人知道珍珍说了什么。

没有人知道她到底摸了谁——或者她到底有没有摸任何人。

但每个人都在想。


第二轮做东是佳佳。

他坐在沙发区的单人座上——左手搭在沙发靠背上右手拿着啤酒瓶。嘉嘉坐在他脚边的地毯上靠着他的膝盖。其他人分布在周围的沙发上。

佳佳选的人是静静。

静静坐在阿敏旁边的双人沙发上——她感觉到自己的脚踝被什么东西碰了一下。她低头——佳佳的脚从茶几下面伸过来了——他的大脚趾碰到了她露在凉鞋外面的脚背。她没有缩回去。

佳佳的脚趾沿着她的脚背滑到脚踝——绕了一圈然后沿着小腿内侧往上滑了一小段停下来。静静的面色如常——她在跟小芳说话。她的声音没有变化。她的小腿肌肉在佳佳的脚趾下微微绷紧了一下然后放松了——她接受了这个触碰。

阿敏在静静旁边刷手机。他没有看到桌下的任何事。

佳佳的脚趾沿着静静的小腿一直滑到膝盖窝——在那里停住了。他的脚趾在她的膝盖窝画了一个小圈。静静的双腿在这时候悄悄打开了几厘米——一个微不足道的距离——但佳佳看到了。

他把脚收回来了。

他说——我做完了。

阿伟——你做了啥。

佳佳——我摸了静静的小腿。

全场安静了。阿敏抬起头——他看了佳佳一眼然后看了静静一眼。静静的表情没有任何变化。佳佳看着阿敏——他说你女朋友小腿很细。

阿敏——你摸了多久。

佳佳——大概两分钟。

阿敏把手机翻过来屏幕朝下放在桌上——消息来了吗。

佳佳——什么消息。

阿敏——你不是说你摸了她两分钟吗。她没让你停？

佳佳——她没拒绝。

阿敏看了看静静。静静端起茶杯喝了一口——她在喝乌龙茶。她放下茶杯的时候杯托发出了一声很轻的碰撞声——不是失手——是她故意放重的。她说——我确实没拒绝。

阿敏沉默了几秒然后说——那你喝三杯吧。猜错了。

佳佳——我猜什么了？

阿敏——你猜她会拒绝。她没拒绝。所以你猜错了。

这逻辑绕了佳佳一脸。但他还是喝了三杯。


第三轮做东是甜甜。

她坐在吧台边的高脚凳上。阿伟在她旁边。其他人基本都在原来的位置——佳佳喝完三杯之后去了厕所。

甜甜选的人是嘉嘉。

她叫了嘉嘉一声——嘉嘉过来一下帮我看看这个。

嘉嘉走过去。甜甜把手机屏幕亮给她看——屏幕上是一张水上乐园的照片。甜甜说——你看这张拍得怎么样。嘉嘉凑过来看手机的时候甜甜的手放到了她的大腿上——隔着嘉嘉穿的佳佳的T恤——她的手沿着她的大腿滑到了T恤下摆的边缘然后伸了进去——手指贴着嘉嘉腰侧的皮肤画了一个圈。

嘉嘉的表情没有任何变化——她说这张拍得不错但有点过曝了。甜甜说——哪张。嘉嘉把手机接过来翻了一下——这张。甜甜的手还在她T恤里面——她的手沿着她的腰线滑到了她的腹部——指尖在肚脐周围画了一个圈。

嘉嘉把手机还给甜甜的时候甜甜的手指在她的乳头位置擦过——隔着内衣的布料——非常轻——像一个不小心。嘉嘉的呼吸停了一瞬。

然后她转身走回了佳佳的座位。她坐下的时候调整了一下坐姿——她的大腿夹紧了一拍然后松开了。

甜甜端起酒杯喝了一口。她说——我做完了。

没有人猜她在做什么。因为没有人看到她做了什么。

点唱机放完了一首歌之后自动切到了下一首——一首老歌。旋律在暗红色的灯光中缓慢流动。窗外的风吹进来把窗帘掀起一角——露出外面漆黑的夜色。

珍珍靠进沙发的深处——她的目光从每个人脸上依次扫过去。甜甜在跟老板说话。阿伟在看她跟老板说话。佳佳从厕所回来在嘉嘉身边坐下——他的手放在她膝盖上。嘉嘉的手覆在他的手背上。静静靠在阿敏肩膀上闭着眼睛。阿敏看着窗外。小芳的头枕在阿贤的大腿上——阿贤的手指在她的头发里。阿强在吧台那边和珍珍隔着一个空位。

老板擦完最后一个杯子把抹布搭在水龙头上。他说——你们这群人挺有意思的。珍珍——哪里有意思。老板——说不上来。就是有意思。

珍珍笑了一下。她从沙发上坐起来——她说最后一轮。我来做东。

她走到吧台最尽头然后转过身。暗红色的灯光从她背后照过来。她看着所有人——她的目光在每个人脸上停了两秒然后用一只手把裙子的肩带拉下来了一边。

她说——我刚才摸了阿强。从裤子伸进去握住了它。但我也摸了另一个人——在同一个时间。你们猜。
