# 毕业旅行 · 第一夜 · 抽签换伴

第一天的晚饭在一家蒙餐馆解决。大桌，中间摆了一个铜锅，里面是手把肉。奶茶是咸的，装在暖壶里自己倒。老板端上来一盆血肠和一盘奶皮子。

气氛比中午自然多了。四年的同学关系在这种时候显示出它的好处——你不需要没话找话。阿伟在用蒙古刀的背面刮羊骨头上最后一层肉，甜甜靠在他肩膀上喝奶茶，佳佳在跟阿敏争论摄影器材的型号，嘉嘉在桌下用脚踢佳佳的小腿让他别喝了。

珍珍坐的位置刚好能看到全桌人。她端着奶茶杯——从她的角度能看到每个人的表情。她注意到阿贤和小芳之间隔了一个座位的距离——不是巧合。因为那个位置上坐着甜甜。

珍珍放下杯子——各位，吃完之后回民宿。今晚没有安排，自由活动。

阿伟抬头看她——怎么个自由法。

珍珍——就是洗澡睡觉。明天还有一整天的活动。

阿伟似乎松了一口气。但珍珍下一句又补上了——当然，如果你不想睡自己的床，也可以。

甜甜在桌下踢了珍珍一脚。珍珍面不改色。


酒足饭饱回到民宿。

夜色已经完全暗下来了。草原上没有城市的光污染，星星多得像有人在天上撒了一把盐。院子里有风，吹得晾衣绳上的床单翻动。珍珍坐在院子里的帆布椅上抽了一根烟，烟头的红光在她手指间一明一暗。

阿强从屋里走出来，在她旁边站了一会儿。他说——外面冷。

珍珍说——那你进去。

阿强没动。他站了一会儿然后说——你今晚和嘉嘉住202是吧。

珍珍——嗯。

阿强——那我就在隔壁。

珍珍抬头看他——你是不是有话想说。

阿强沉默了几秒——没有。

珍珍把烟掐灭在烟灰缸里——那就别说。


房间的分配在晚饭前就已经定了。此刻十个人分布在民宿的三间房和公共区域里。

202住的是珍珍和嘉嘉。佳佳和阿强住在隔壁203。两间房的阳台之间只隔了一堵不到膝盖高的矮墙。

嘉嘉先洗的澡。她出来的时候裹着浴巾，头发湿漉漉地贴在脖子上。珍珍靠在床上刷手机，抬头看了她一眼——你身材不错。

嘉嘉笑了一下——你第一次看？

珍珍说——第一次认真看。

嘉嘉擦着头发走到窗边。她拉开窗帘看了一眼外面的草原——什么都看不见，因为太黑了。她拉了拉窗帘的边角然后转身——你猜阿伟今晚能不能憋住。

珍珍放下手机——什么意思。

嘉嘉——201是三张床。一张大床两张小床。甜甜和阿伟睡大床，阿贤和小芳各睡一张小床。

珍珍——所以。

嘉嘉——但甜甜已经告诉我了，她今晚的目标不是阿伟。

珍珍坐直了一点——是谁。

嘉嘉没有直接回答。她坐在另一张床的床沿上，面对着珍珍。浴巾的边缘在她大腿根部微微张开——她没有合拢腿。她说——你猜。

珍珍没有说话。她看着嘉嘉的眼睛。浴巾下面的阴影在灯光下显得很深。

嘉嘉说——反正不是佳佳。佳佳今晚的目标也不是我。

珍珍——佳佳今晚的目标是谁。

嘉嘉——甜甜。


201的灯开着。甜甜趴在床上玩手机，两条腿翘起来在空中晃。她穿着一件宽大的白T恤当睡衣，领口大得能看到锁骨的延伸线和胸口的部分弧度。阿伟坐在单人床的边缘——他把那张小床让给了阿贤，自己坐在那里像是不知道该做什么。

阿贤在卫生间刷牙。水声哗哗的。

甜甜头也不抬——阿伟，你去洗澡吧。

阿伟站起来的时候裤裆那里鼓起来一块。甜甜看到了，但假装没看到。她继续刷手机，等卫生间的水声停了、阿贤出来、阿伟进去、卫生间的门关上之后，她才把手机放下。

阿贤擦着头发走出来。他只穿了一条运动短裤，上身赤裸。他平时穿衣服看不出来——但脱了之后肩膀的宽度和腰的对比很显眼。腹肌不是那种健身房雕出来的六块——是打球打出来的线条，不夸张但结实。

甜甜看着他从她面前走过。她的目光从他的肩膀滑到他的腰，然后停在那条灰色运动短裤的某个位置。她没有移开目光。

阿贤感觉到了。他在床边停下，侧头看她。

甜甜——你身材比我想象中好。

阿贤——你想象过。

甜甜——嗯。

阿贤没有接话。他坐在那张小床上，背靠着墙。他的小腿太长，脚踝伸出了床沿。两个人之间隔了不到两米的距离。

卫生间的淋浴声哗哗的。阿伟在里面。

甜甜翻身改成趴着的姿势，下巴搁在枕头上。她的领口因为这个动作垂下去——从阿贤的角度能看到她整个乳房的侧面轮廓。她没有穿内衣。

甜甜——你觉得这趟旅行会怎么收场。

阿贤——什么意思。

甜甜——我的意思是——七天后我们回到学校，这七天发生的事，到底算不算数。

阿贤——你希望算不算数。

甜甜没有回答。她翻了个身仰面躺着，看着天花板。卫生间的淋浴声停了。她坐起来整理了一下T恤的下摆——动作很快，像什么也没发生过。


阿伟从卫生间出来的时候只围了一条浴巾。他比阿贤矮一些但比他壮——胸肌和肩膀都练得很厚。浴巾围得不高，露出小半个腹部和人鱼线的起始位置。

甜甜看了他一眼——你最近是不是练得挺好的。

阿伟——嗯。

甜甜拍了拍床垫——过来。

阿伟走过去坐在床边。甜甜从床上坐起来，膝盖跪在床垫上，比坐着的阿伟高出一截。她低头看着他——浴巾的边缘已经有一点撑起来了。她伸手碰了碰那个位置——指尖沿着撑起的轮廓画了一道弧线。

甜甜——你今天不怎么高兴。

阿伟——我没有。

甜甜——你有。因为抽签的事。

阿伟没说话。

甜甜跪在他面前——她的手没有离开那个位置。她说——你是不是怕我跟别人做什么。

阿伟——不是。

甜甜——撒谎。

她拨开浴巾的边缘，低头含住了他。阿伟的呼吸在那一瞬间变得很粗。甜甜的动作不快——她慢慢地、深深地含进去，舌头绕着他的龟头打转，然后在喉咙深处停留了两三秒才退出来。

她抬起头——嘴角牵着一丝唾液。她说——现在好一点了吗。

阿伟没有回答。他把甜甜按倒在床上——她的T恤被推到锁骨以上，一对不算大但形状很漂亮的乳房弹了出来。他俯下身含住了其中一颗乳头。甜甜发出了一声低低的哼声。

单人床那边传来一声轻响——阿贤翻了个身，面朝墙壁。

阿伟的动作停了一下。甜甜的腿缠上他的腰——别停，她说——不用管他。


那天夜里202发生的事不太一样。

嘉嘉躺在大床的左边。珍珍躺在大床的右边。两个人中间隔着一个人的距离。灯已经关了。窗外的月光从窗帘的缝隙里漏进来。

嘉嘉开口——你睡了吗。

珍珍——没有。

嘉嘉——你觉得阿强今晚会不会来找你。

珍珍——他不会。

嘉嘉——为什么。

珍珍——因为他怕敲门之后发现房间里不止我一个人。

沉默了几秒。嘉嘉翻了个身面朝珍珍的方向——那你希望他来吗。

珍珍没有立刻回答。她也在黑暗中翻了个身，她们之间的距离从一个人的距离变成了半个手臂的距离。

珍珍——你今天的问题还没回答我。

嘉嘉——什么问题。

珍珍——佳佳今晚的目标是谁。

嘉嘉——甜甜。

珍珍——那你的目标是谁。

嘉嘉的脚在被子里碰到了珍珍的小腿。她的脚趾沿着珍珍的腿往上爬了一小段——然后停住了。

嘉嘉——你。

珍珍没有躲。她的呼吸在黑暗中变快了一点点但幅度很小。她说——我们都在谈恋爱。

嘉嘉说——所以呢。

所以没人再说话了。她们在黑暗中安静地躺着，距离半个手臂，嘉嘉的脚趾还搭在珍珍的小腿上没有移开。


半夜两点左右，201的灯已经全熄了。

大床上阿伟已经睡着了——呼吸声很沉很均匀。甜甜背对着他侧躺着，面朝窗户的方向，眼睛睁着。

单人床上阿贤面朝天花板躺着，也没有睡着。

甜甜悄无声息地从大床上坐起来。她赤着脚踩在木地板上——没有发出任何声响。她绕过床尾走向门口。经过阿贤的单人床时她停了一下——她的手指在黑暗中碰到了他的手指。轻轻一碰。然后她继续走向门口。

门开了又合上。咔哒一声很轻。

阿贤躺了几秒钟。然后他也坐了起来。

走廊的声控灯被她的脚步点亮了一盏。他看到甜甜穿着那件肥大的白T恤站在走廊尽头201的门口——她在等他。

她没有说话。她伸手推开了203的门——那是阿敏的房间。

阿敏不在房间里。静静也不在。他们的阳台门开着——那是通向二楼公共露台的门。露台上隐约有人影——两个。阿敏和静静坐在露台的椅子上看星星。

甜甜没有进203。她回头看了阿贤一眼。然后她走进了203隔壁的门——那是202。

珍珍和嘉嘉的房间。

阿贤站在走廊里。灯灭了。他没有跟进去。

他转身回了201。


早上七点。草原的日出很早，光线从窗帘的缝隙里挤进来，把地板染成橙黄色。

阿贤醒的时候发现小芳不知道什么时候从他的单人床旁边——她坐在地板上，背靠着他的床沿，在系鞋带。她穿着一件白色的速干T恤和黑色短裤，头发扎了起来。

她说——早。

阿贤——你什么时候过来的。

小芳——你睡着之后。床太硬了。

阿贤——那你怎么不叫我。

小芳系好鞋带站起来——因为有人昨晚不在自己床上。需要我叫你吗。

阿贤没有说话。

小芳拍了拍他的膝盖——早餐七点半。别迟到。然后她走出了房间。

阿贤躺在床上看着天花板。窗帘被风吹起来又落下。隔壁传来阿伟翻身的声音——他还睡着。

早餐桌上是比较沉默的。奶茶、炸果子、和一盘切成块的奶酪。

每个人都在。十个人。甜甜坐在阿伟旁边，一边喝奶茶一边和佳佳讨论今天去草原的路线。她的表情和平时没有任何区别。嘉嘉坐在珍珍对面，两个人之间隔了一盘奶酪，谁也没有看谁。

阿敏在帮静静夹奶茶里的奶皮子——他的手指很长，拿筷子的姿势很好看。静静接过碗的时候说了声谢谢。

珍珍咽下嘴里的奶茶——今天去草原。蒙古包已经订好了。下午回来泡温泉，晚上篝火。

阿伟咽下一口奶茶——有什么安排。

珍珍——明天你自己看。

第一夜结束了。没有发生任何"实质性"的事——如果身体的物理接触不算的话。但所有人都知道，这只是一个开始。因为规则才刚刚开始松动，而距离旅行结束还有六天。
