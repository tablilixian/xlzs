# 毕业旅行 · 第二天深夜 · 温泉水下暗战

从草原回到民宿是晚上八点多。所有人都被草原的风吹透了——身上裹着一层细沙和汗混合的尘土。珍珍宣布了一个消息：民宿后面有一个露天温泉，男女混浴，刚到的热水。想去的人自己去，不去的先洗澡睡觉。

没人说不去。

十个人回到各自的房间换了衣服披上浴袍走向后院。温泉在院子最深处的角落里——被一圈竹篱笆围着，篱笆外面种了一排不知道名字的灌木。池子不大——直径大概四米左右，天然石砌的，池底铺了卵石。水很热——热气蒸腾上来像一锅刚揭盖的汤。水面上漂浮着一层白色的蒸汽，浓到三米之外就看不清人脸。池边插了几根竹竿，顶上挂着几盏防水的灯笼——橘黄色的光穿透雾层变成一团一团模糊的光晕。

十个人下水。

池子刚好能坐下十个人——肩膀挨着肩膀，大腿碰着大腿。水面到每个人的下巴，有人矮一点水没到嘴角，有人高一点露出锁骨。热水从脚底涌上来——整个池子像一口巨大的汤锅，十个人在里面慢慢煮熟。

珍珍第一个脱了浴袍挂在篱笆上。她穿了一套黑色的比基尼——两根细带挂在肩膀上，胸口的布料少到刚好盖住乳尖。她滑进水里的时候发出一声满足的叹息——操，太爽了。

嘉嘉第二个——她直接穿了一件白色的运动内衣和一条深色的紧身短裤下水，但下水之后她把内衣脱了团成一团扔在池边——她说穿着泡没意思。她的乳房在水面下若隐若现——乳尖在水波中露出又沉没。

甜甜穿了一件白色的两截式——泳裤的带子在髋骨两侧各打了一个蝴蝶结。她踩着池边的石头走进水里的时候所有人都看到了她腰两侧那两道蝴蝶结——轻轻一拉就能解开。

静静穿了一件深蓝色的连体泳衣——保守款但她的身材让任何泳衣都保守不起来。她下水之后坐在池子的左侧，水没过她的肩膀，只露出头和锁骨的上沿。她用手把湿头发拢到脑后——动作慢得像在拍广告。

小芳穿了一件浅灰色的比基尼——款式比珍珍的还保守一点但她的肤色很白，在雾气里白到发光。她下水之后坐在阿敏旁边——不是阿贤旁边。阿贤在池子的另一端。她入水之后没有看他。

阿强第一个脱浴袍下水的男的——他穿了一条深灰色的泳裤，下水之后坐在珍珍旁边。佳佳穿了一条荧光绿的泳裤——被所有人嘲笑了。他说这条是嘉嘉给他买的，嘉嘉说好看。阿伟穿了一条黑色的平角泳裤，下水的时候肌肉绷了一下。阿敏穿了一条浅蓝色的沙滩裤。阿贤穿了一条深色的运动短裤——下水之后他坐在池子的最右端，靠着竹篱笆。

十个人围成一圈坐下。水面蒸汽升腾。灯笼的光在雾气中化成一团模糊的橙色光晕。没有人说话——头几分钟只有热水翻涌的声音和偶尔有人调整坐姿带起的水声。

然后雾变得更浓了。


池子里的第一个动作发生在水面下。

蒸気越来越浓，浓到坐在池子对面的人只能看到一个模糊的头部轮廓。池底的卵石被热水泡得发烫，脚踩在上面有一点硌。水面下是另一个世界——一个与上面完全无关的世界。水面上大家在聊天、在笑、在仰头看星星。水面下什么都有。

阿贤靠坐在池壁的角落，热水齐到他的锁骨。他闭着眼睛，肩膀沉在水里。他感觉有人从水下靠近——水波的变化比声音更快。有人的膝盖碰到了他的大腿外侧。他没有睁眼。

那人继续靠近——一条光滑的腿跨过了他的大腿。水的阻力让那个动作比平时慢了一倍。他睁开眼。甜甜正对着他跨坐在他的大腿上——水面刚好没过她的腰部，从水面上看，她只是换了位置，面朝他坐着而已。但水面下——她的腿分开了，泳裤的蝴蝶结已经被解开，她光着的臀部压在他的大腿根部，那层薄薄的泳裤布料隔在她和他的运动短裤之间。

甜甜看着他，笑了一下——这个位置暖和。

她沉下去了一点——水没过了她的胸口。她的手在水下找到了他的运动短裤前面的隆起——已经硬了。她用指尖隔着布料描了一遍他的形状。

甜甜——泡温泉是不是容易让人想一些不该想的事。

阿贤——你已经做了不该做的事了。

甜甜笑——这才刚开始。

她沉进水里。水面上只剩一串气泡。三秒——五秒。她在水下做了什么只有她自己知道。她浮出水面的时候抹了一把脸，嘴角有一点水痕。她喘了一口气——然后跨坐得更实了。

阿贤的手从水下伸过来握住了她的腰。他把她往自己的方向拉了一下——她的臀部结结实实地压在了他的裆部。隔着两层湿透的布料她能感觉到他的硬度——完整的、坚硬的轮廓贴在她的阴部上，隔了两层薄薄的织物。

甜甜低下头，嘴唇凑到他的耳边——热气喷在他的耳朵上。她说——阿贤，你知道吗，据说男女上下连通，就能形成一个能量循环，在这个温泉中，更能增强身体的能量。

阿贤没有说话。他转头吻住了她——不是那种浅尝辄止的吻。他的舌头直接撬开了她的牙关，一口把她的舌头吸进了自己的嘴里。她发出了一声被堵住的闷哼——嗯——他的右手从水下抬起来握住了她的乳房。白色的比基尼布料在水面上若隐若现——他的手指包住了整个乳房的轮廓，拇指压着乳尖的位置揉了一圈。甜甜的嘴唇从他嘴里滑出来——她咬着自己的下唇不让自己出声。他的左手也从水下抬起来——他找到了她另一边的乳头，用拇指和食指捏住，忽轻忽重地搓揉。甜甜的呼吸变得又急又碎——她趴在阿贤的肩膀上，嘴唇贴着他的脖子，压抑的呻吟从嘴角断断续续地漏出来——嗯……嗯……

她缓过一口气，嘴唇贴到他耳边——小芳就在对面。你老婆就在对面看着。

阿贤没有说话。他托着她的腰把她往上抬了一点——她重新坐下去的时候他的龟头隔着两层布顶在了她的阴唇之间。甜甜的瞳孔在那个瞬间放大了一倍——她咬住了自己的手背。

水波从他们身上荡开——扩散到池子中央。


珍珍坐在池子的另一侧。

她的左腿曲起来踩在池沿上，右腿伸直。阿强坐在她旁边——他的右手放在她的大腿上，拇指在她膝盖内侧画圈。珍珍在和阿敏聊天——聊他的摄影器材和这次旅行要拍的东西。

水面下阿强的右手正在往她大腿根部移动。他的手指沿着她的比基尼泳裤的边缘滑进去——那层布料已经被水浸透了，薄得像一层皮肤。他的中指碰到了她的阴蒂——她已经硬了。她戴着水下戒指的那根手指指甲修剪得很整齐——他的指尖绕着那颗小豆转了一圈。

珍珍的表情没有任何变化。她继续和阿敏说话——她说你这次拍了多少张了。阿敏说大概三百多张。珍珍说晚上看看。阿敏说可以。

水下面阿强的手指已经滑进了她的泳裤里面。他的中指沿着阴唇的缝隙滑进去——她里面又热又滑。他慢慢推进去——一根手指的全部长度。珍珍的呼吸在那一瞬间断了一拍但她立刻接上了——她说你拍静姐拍得最多吧。

阿敏笑了一下——被你发现了。

阿强的手指在她的阴道里慢慢进出——水面下的动作和上面的话题完全无关。珍珍的脚趾在卵石上蜷缩又松开。她的表情平静——甚至带着一丝笑意。水面下阿强正在用手指操着她，动作匀速，不急不缓，拇指同时按着阴蒂画圈。

珍珍的呼吸变得越来越浅。她把手伸到水下——她握住了阿强的手腕不让他抽出来。她让自己的屁股往前送了一点——让他的手指进得更深。她咬着嘴唇——在阿敏说完一句话的间隙里嗯了一声作为回应。

没有人发现。除了佳佳——佳佳坐在对面的位置。池子的直径对他来说不算什么——他恰好看到了珍珍那一瞬间的表情。他没有说话。他低头喝了一口水。


静静坐在池子的左端。

她靠在池壁上，双臂搭在池沿，头仰着看天上的星星。雾气在她面前的池面上翻滚。她的身体沉在水面下——深蓝色的泳衣包裹着她那一对引人注目的乳房，热水托着她的身体让她几乎感觉不到自己的重量。

阿敏不在她身边——他坐在池子的另一端和珍珍聊天。她的旁边坐着阿伟。

阿伟也在看星星——或者假装在看星星。他的手从水下伸过来碰到了静静的大腿外侧。第一次碰到的瞬间他以为她会躲开。她没有。

他的手指在她的大腿外侧停留了一会儿——像是不确定下一步该做什么。静静的呼吸平稳，她的目光停留在天上的星星上没有移开。她的沉默本身就是一种许可。

阿伟的手沿着她的大腿外侧滑到了内侧。她的皮肤在水里烫得很滑——热水让触感变得不太真实。他的手指碰到了泳衣的边缘——深蓝色的布料包裹着她的阴部。他的手覆在那上面——隔着泳衣她感受到他的掌心的温度和压力。

静静依然没有动。她的胸部在水面下起伏的幅度变大了——唯一的信号。

阿伟的手指隔着泳衣按了一下。她的大腿几不可察地分开了一点点——只有他能感受到的那一点点。他的手指沿着泳衣的边缘滑进了一根手指的宽度——指尖碰到了湿润的、柔软的、比池水热得多的区域。

静静的嘴唇张开了一条缝。她没有发声——但她的身体往他的方向偏移了几厘米。那几厘米足以让他的手指进入得更深。

——阿伟？

甜甜的声音从池子的另一端传过来。

阿伟的手瞬间收了回来。他转头——甜甜还坐在阿贤身上——她看着他，眼神意味深长。

阿伟——嗯？

甜甜——你那边热吗。

阿伟——还行。

甜甜——那就行。别泡太久。

她说完又转回去了。阿伟低头看着水面——水波晃动，什么也看不见。他的手还留着湿润的触感。静静在看星星——她的嘴唇还是微微张开着。


嘉嘉在池子里换了好几个位置。

一开始她坐在佳佳旁边——但佳佳一直在跟阿强喝酒聊天，她听了一会儿觉得无聊就移到了池子的另一侧。她靠着池壁半躺着，热水淹到她的嘴角。她的眼睛半闭着看起来像快要睡着了。

但她的脚没有睡着。

她的脚趾在水下沿着池底慢慢移动——碰到了一个毛茸茸的小腿。她用脚趾沿着那根小腿的胫骨从下往上滑动。那个人没有躲。

她把脚收回来几秒然后又伸过去——这次直接搭在了那人的大腿上。她的脚趾在那人的大腿根部轻轻按压——那个人硬了。她的脚趾沿着硬起的轮廓从根部滑到龟头再滑回来——隔着泳裤那层布料她能感受到温度和形状的变化。

那个人一直没说话。她也没说话。她的脚继续在水下活动——她勾住了泳裤的边缘试图往下拉。

那个人握住了她的脚踝——轻轻地，但坚定地。

然后她听到了阿强压低了的声音——这个不行。

嘉嘉笑了——她收回脚，在水池里翻了一个身。她说无所谓——然后游到了池子的另一端。


雾气越来越浓了。

灯笼的光被浓雾压成了几团模糊的光晕。伸出手臂就看不清手指。水面上的人影只剩下头部和肩膀的轮廓——混在蒸汽里几乎无法辨认谁是谁。

然后一阵大雾滚过来——像有人在天上倒了一锅干冰。整个温泉池子被白色的浓雾完全吞没了。三秒之内你连面前的人的脸都看不清。

池子里安静了一拍。然后是水花翻动的声音——不止一处。

甜甜从阿贤身上滑下来。她牵着阿贤的手穿过浓雾走到池子最深处的角落——那里的水齐到胸口，阴影浓到即使雾散了也看不清。她背对着他弯下腰扶着池壁的石头——白色的臀部浮出水面，水珠沿着她背部的曲线往下滚落。她回头看他——她看不清他的脸但她知道他在什么地方。

快。她说——趁雾没散。

阿贤的手从水下伸过来握住了她的胯骨。他的另一只手扶着阴茎——龟头碰到她阴唇的时候两个人都停了一下。她回过头趴在池壁上——她在等他。

他沉腰。龟头撑开了她的阴唇——她里面比池水热得多——他完整地一插到底。她发出了一声被压在喉咙深处的闷哼——额头抵在自己交叠的手臂上。她的阴道包裹着他的全部——紧、热、还在收缩。

他没有马上动。

甜甜的声音在雾气中闷闷地传过来——你动啊。

他抽出来大半然后重新顶进去。噗——水声被雾气吸收了。咕叽咕叽的声音从水面以下传上来——被热水的咕嘟声掩盖了一半。

甜甜咬着自己的手背。她的身体随着他的冲击一下一下地向前滑动——她的手撑在池壁上，指节发白。她的乳房随着撞击在水面下晃动——白色的泡沫在他们结合的地方扩散开来被热水冲散。

又一阵大雾从池面上方压过来——比刚才那阵更浓。整个温泉完全变成了白色的深渊。十厘米之外什么都看不见。

甜甜转过头——她的嘴唇贴在他的耳边——操我——她用气音说——用力操我——不要让小芳知道你会这样操别人。

阿贤握紧她的腰加快了速度。啪啪啪——她的臀部被他撞得发红，水花四溅。甜甜咬着自己的拳头——呻吟被堵在喉咙里变成了闷闷的呜呜声。她的阴道突然绞紧——她高潮了——她趴在池壁上整个人都在发抖。

远处传来一阵车的引擎声。一道车灯从篱笆上方扫过来——刺眼的白色光束切开了雾气。那一瞬间——不到两秒——灯光照亮了整个温泉池子。

所有人都看到了所有人。

甜甜弯腰趴在池角的石壁上，阿贤站在她身后，她的臀部顶着他的胯部——两个身体连在一起。珍珍坐在池边双腿大张，阿强跪在她面前头埋在她腿间——水面只没过他的膝盖。佳佳靠在池壁上，嘉嘉坐在他腿上上下起伏——水花从他们结合的位置一圈一圈地荡开。静静站在池子的另一端，阿伟站在她面前——她的一只手撑在他的胸口，另一只手在水下握着他的阴茎但没有放进去。阿敏靠在不远处的池壁上——他的泳裤半褪到臀下，但他面前没有人。

灯光过去了。黑暗重新降临。

没有人停下。

车灯消失之后池子陷入了几秒钟的绝对黑暗和安静。然后不知道是谁先发出的声音——可能是一声压抑的笑，可能是一声不受控制的呻吟。然后水声又响起来了。所有人都知道所有人都看到了彼此——正因为如此，没有人需要解释任何东西。

雾又聚拢过来——像一层新的幕布盖在舞台上面。


池子中央的雾气暂时薄了一点。

佳佳和嘉嘉没有移动位置。从刚才灯亮的时候他们就没分开过——或者说他们分开了也没有人能确认。佳佳坐在池边的石阶上，水只没到他的腰部。嘉嘉跨坐在他身上——白色的运动内衣不知道什么时候已经被丢在了池边，她的乳房裸露着贴在他的胸口。她的腰在慢慢摆动——不快，不急，每一下都坐得很深。她的手插在他的头发里，额头抵着他的额头。

佳佳——你刚才看到了吗。

嘉嘉——看到什么。

佳佳——阿敏。

嘉嘉——他一个人站在那边。

佳佳——对。

嘉嘉——你想说什么。

佳佳——他什么都没做。他本来就是来看的。

嘉嘉的腰停了一下然后又继续动——那你还看。

佳佳的手握住她的腰帮她调整了一下角度——不看了。我看你就够了。

嘉嘉低头吻他。水面下的动作没有停。


阿强被车灯光照到的时候他正在做的事情被所有人看到了——他的脸埋在珍珍的腿间，嘴唇含着她最敏感的位置，舌尖正在她的阴蒂上画圈。灯光亮了不到两秒——但那两秒里珍珍能看到阿强的后脑勺在自己腿间，能看到对面佳佳和嘉嘉交叠的身影，能看到静静和阿伟面对面站着阿伟的手在水下。

灯光灭了之后阿强没有抬头。他的舌头继续在同一个位置活动——节奏没有变化。珍珍的手按在他的后脑勺上——她把他的脸更紧地压向自己。她的阴道在他的嘴唇前完全打开——她不在乎谁看到了。

她也不在乎阿敏刚才一个人站在池子对面的阴影里——她不在乎他是不是在看着自己。她甚至不在乎他是否从刚才灯光亮起的时候就一直在看。反正明天没有人会提这件事——这就是规则。


静静站在深水区。

她听到了周围所有的水声——她右侧传来的有节奏的拍打声，对面珍珍压抑的喘息声，池角那对撞击水面的声音。她的脸在黑暗中红到了脖子根。

阿伟还站在她面前。他从刚才灯亮之后就一直没有动——手还放在水下的位置但没有进一步的行动。像在等她决定。

静静伸出了手。她的手在水下碰到了他——他的泳裤前端鼓起的弧度很大。她隔着泳裤握住了他——她听到了阿伟的呼吸一下子变粗了。她犹豫了一下。然后她把他的泳裤边缘往下拉了一点——那根阴茎弹出来，碰到了她的手背。她握住它的时候掌心碰到了龟头——很烫，比池水烫。

她抬头看了他一眼。雾太浓了她看不清他的脸。

她沉入水中——水面没过她的头顶。她在水下张开了嘴。

阿伟的呼吸猛地抽紧。他的手指攥紧了池壁的边缘。水面上安静得像什么也没发生——只有一串细密的气泡从某个位置持续地浮上水面，在雾气中破裂，又被新的气泡取代。

甜甜趴在池角喘够了气。

她从水里站起来——大腿内侧有水珠在往下流，混着白色的液体被池水冲散。她走到池边坐下，双腿垂在水里。阿贤也走过来坐在她旁边。两个人在雾气中并肩坐着，肩上披着同一条浴巾——不知道是谁拿过来的。

甜甜——你说她知不知道。

阿贤——谁。

甜甜——小芳。

阿贤沉默了一会儿——她知道。

甜甜——那她为什么不拦着。

阿贤——因为这趟旅行每个人都在做选择。

甜甜笑了一下。她靠在他的肩膀上——就十分钟。没人会知道。

阿贤没有说话。雾在他们面前翻滚。池子里其他人的水声还在继续——水花拍打、喘息被压制、压抑的呻吟在雾中此起彼伏。

远处又传来一阵引擎声。车灯扫过篱笆——光束穿过雾气照亮了温泉池。这一轮灯光比刚才那盏更亮——像一辆大车正在靠近。

灯光照亮了整个池子。

阿敏还站在他原来的位置——他面前多了一个人。小芳站在他面前。她的比基尼上半截被解开了挂在她的一侧手臂上——她的乳房裸露着贴在他的胸口。她的嘴唇贴着阿敏的嘴唇但两个人没有在接吻——只是贴着。

阿敏的手放在她光裸的后背上。没有进一步的动作。他们只是站在那里——像一尊双人雕塑。

灯光过去了。池子重新陷入黑暗。

小芳从阿敏身上退开。她把比基尼的带子重新系好——动作很从容。她没有看阿敏的脸。她也没有看池角的方向——她知道阿贤和甜甜并肩坐在那里。她走到池子的另一侧滑进水里。

她闭着眼睛泡了很久很久。没有人打扰她。

凌晨一点左右温泉里的人开始陆续散了。

没有人说"走了"——一个人起身披上浴袍离开，过了一会儿另一个人也起身离开。像一场没有谢幕的演出——演员一个一个退场，舞台上的灯一盏一盏熄灭。

珍珍和阿强是最早走的两个——不是一起走的，但前后相差不到两分钟。

静静和嘉嘉几乎是同时离开的——她们在池边碰了一下目光然后各自裹上浴巾往回走。

甜甜站起来的时候腿软了一下——阿贤扶了她一把。她拍了拍他的手说不碍事。她裹上浴袍走了几步然后回头——她说阿贤。

阿贤——嗯。

甜甜——今晚是今晚。明天是明天。

她走了。阿贤还坐在池边。雾开始散了——灯笼的光重新变得清晰。小芳从池子的另一端起身。她走过他身边的时候没有停下脚步。

阿贤——小芳。

她停下了。

阿贤——你需要解释吗。

小芳背对着他站了几秒钟。她说——不。然后她继续走了。

池子里最后一个人是阿敏。他坐在池沿上看着水面——水已经因为所有人的离开而安静下来了。灯笼的倒影在水面上微微晃动。他不知道自己在那里坐了多久然后站起来披上浴袍走回房间。

他推开门的时候静静还没有睡。她靠在床头看手机。浴室的灯亮着——阿敏以为她在等他。

静静头也不抬——浴室的水是热的。

阿敏站在门口。浴袍还湿着贴在身上。他说——你都看到了。

静静放下手机——对。

她说——我不在意你跟她做了什么。

她翻了一个身背对着他——我只在意你回来的时候有没有在想着她。

阿敏站在门口没有动。浴室的热水还在流——水汽从门缝里涌出来。
