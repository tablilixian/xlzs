# 毕业旅行 · 开篇

车是珍珍从她表哥那里借来的九座商务车。后排塞了五个行李箱和几箱饮料，最后一排放倒铺成了一个小型的"休息区"——铺了毯子，堆了几个靠枕。阿贤坐在驾驶座上调整后视镜，珍珍坐在副驾，脚翘在仪表台上。剩下八个人挤在后面几排，有人已经脱了鞋把腿伸到旁边的座位上。

出发时间是早上六点半。天刚亮透，城市还没醒，高架上没什么车。

阿贤看了一眼后视镜——小芳坐在第三排靠窗的位置，戴着耳机，头靠在窗户上。她感受到他的目光似的抬起头，在后视镜里和他对视了一秒，然后笑了一下。

旁边的阿伟在大声抱怨空调不够冷。

甜甜靠在阿伟肩膀上，手指在他胸口画圈。她说你少穿一件就不热了。阿伟说你帮我脱。甜甜笑着说想得美。

中间那排坐着静静和阿敏。静静也在听歌，闭着眼睛，阿敏在拍窗外的风景——他的相机镜头时不时转过来拍一张车里的动态。他拍珍珍翘在仪表台上的脚踝，拍甜甜靠在阿伟肩上的侧脸，拍佳佳在最后一排用手机外放打游戏的声音。

佳佳在打王者，骂队友的声音大得全车都能听见。嘉嘉坐在他旁边，戴着一边耳机看他打，偶尔伸手戳一下屏幕说"你从这边绕过去"。

七点整，车上了高速。珍珍从她那个巨大的托特包里掏出一把用橡皮筋扎着的纸条。

她说——先说明，这趟旅行，所有的房间分配、床位安排、搭档组合，全部抽签决定。

阿伟第一个说话——凭什么。

珍珍回头看他——凭车是我借的，油是我爸加的，民宿是我订的。你有意见？

甜甜拉了拉阿伟的衣角说算了。阿伟闭嘴了。

珍珍把纸条分成两把——男生抽一把，女生抽一把。男生抽的是今晚的"室友"，女生抽的也是今晚的"室友"。

规则是这样的：每天晚上抽一次，抽到谁的名字，今晚就和谁睡一屋。但房卡只有一张，也就是说——同一间房里谁进谁出，没人管。

阿贤先抽。他打开纸条的时候表情没什么变化，但坐在第三排的小芳看到他把纸条迅速折起来塞进了口袋。

阿伟第二个抽。他看了一眼纸条上的名字，脸色变了一下又恢复。他问珍珍——这是什么意思。

珍珍说——字面意思。

甜甜好奇地凑过来要看阿伟的纸条，阿伟躲开了。他说没什么好看的。

珍珍自己也抽了一张。她看了一眼，笑了一下，然后把纸条夹进了手机壳后面。

所有人抽完之后，珍珍拍了拍手——好，从现在开始到今晚入住，谁都不许提前通融或者交换。抽到谁就是谁，这是规矩。

这条规矩持续了不到十个小时。


这十个人同班四年，但在这趟车之前，没有人在同一个空间里待过这么久。

小芳是班花。不是那种挂历上的班花——她长得不算惊艳，第一眼看是舒服，第二眼看是好看，第三眼看就移不开了。她皮肤很白，头发是那种自然垂顺的黑长直，平时上课喜欢穿宽松的针织衫和牛仔裤，但所有人都知道她身材很好——大一体育课八百米测试，她穿了一件白色紧身T恤跑了全班女生第一，胸口的起伏让当时在场的大半个班都沉默了。她说话声音不大，但每句话都很到位。她从不主动惹事，但也不怕事。追了她四年的男生不少但没一个追到过——直到大三那年阿贤不知道用什么手段把她拿下了。

阿贤是校篮球队的，打得分后卫。一米八五，肩膀宽，手臂上有肌肉但不夸张。他不算传统意义上的帅哥——单眼皮，鼻梁高，嘴唇偏薄，笑起来有一点痞。他在场上打球的时候场边永远有女生看，但他从来没跟任何女粉丝走得过近。他跟小芳在一起之后很多人都意外——不是意外他们在一起，是意外阿贤居然会定下来。他平时话不多，但在某些场合话会突然变多，而且专挑让人脸红的说。

珍珍和阿强是那种让所有人看不懂的组合。珍珍是班里有名的辣妹——短裙、低胸、大红唇，学生会外联部部长，什么活动都敢上台，什么酒都敢喝。她的社交账号粉丝比全系其他人加起来都多。阿强是体育委员，校足球队的，壮实，晒得黑，平时在班里不怎么说话，但笑起来很憨。没人知道珍珍为什么选了阿强——有人猜是阿强某个方面特别强，珍珍从来没否认过这个猜测。珍珍在公开场合对阿强呼来喝去，动不动就骂他木头，但阿强从来不还嘴。有人说他是怕珍珍，有人说他是真的喜欢她所以让着她。

甜甜和阿伟是另一类典型。甜甜长了一张人畜无害的脸——圆眼睛，小酒窝，说话尾音往上翘，喜欢穿粉色和白色的衣服。第一次见她的人都觉得这是个乖乖女。认识她超过一个月的人都知道她玩得比谁都花。她会在朋友圈发抱着布偶熊的照片，然后半夜在酒吧卡座上和一个刚认识十分钟的人舌吻。阿伟是她的高中同学，在一起三年了。阿伟占有欲很强，刚在一起的时候因为甜甜和别人多说几句话就会生气。但甜甜有办法治他——她会在和他吵架之后跪下来拉开他的拉链。久而久之阿伟的态度变了。现在甜甜在派对上和别人调情的时候阿伟会在旁边看着，勃起，然后等回家之后狠狠地操她。

静静和阿敏是所有人都觉得"早晚会分但一直没分"的那种情侣。静静在班里几乎不说话。她不是那种刻意高冷的安静——是真的不爱说话。但她坐在那里的时候没人能忽略她。一米六八，D罩杯，腰细得一只手就能圈住，腿又直又长。她有一张让人过目不忘的脸——五官偏冷，不爱笑，但笑起来的时候整个人会变一个样。阿敏是摄影社的社长，追了她两年。他话也不多，但会用镜头表达——他的相机里百分之七十的照片都是静静。他拍她读书的样子、吃饭的样子、靠在窗边发呆的样子。他也会在旅行中把相机递给别人，让别人拍他和静静的合照——但接过相机的人总能从那取景框里感受到一种微妙的占有欲。他喜欢别人看静静，但他更喜欢那种"你们只能看，她是我的"的感觉。

佳佳和嘉嘉是大四才在一起的。佳佳家里有钱，不是一般的有钱——他爸在本地有三家公司。但他从来不炫富，穿得比谁都随便，唯一的奢侈品是那双限量版的球鞋。他性格好到几乎没脾气，什么都无所谓，什么都行，唯一在乎的事情就是"玩得爽不爽"。他喜欢组织活动、攒局、买单。嘉嘉是学妹，小他们一届，是佳佳在社团活动上认识的。她看着文文静静的，戴着一副圆框眼镜，说话声音不大，第一次跟这伙人吃饭的时候全程没怎么抬头。但第二天佳佳就跟朋友说——这女的比你们所有人都猛。没人信。直到有一次大家玩真心话大冒险，嘉嘉抽到了"跪着用嘴开啤酒瓶"，她二话不说跪下去——用时七秒。全场安静了三秒然后炸了。


车开了四个小时之后下了高速，进入省道。路两边从楼房变成了田地和零星的白杨树。天很蓝，云很低。

阿贤在服务区停了车让大家上厕所。珍珍跳下车伸了一个大大的懒腰，T恤下摆提起来露出一截腰。阿强从后面走过来的时候看了一眼，然后迅速移开目光。

甜甜拉着小芳去便利店买烤肠。小芳站在冰柜前面选饮料的时候甜甜靠在货架上看着她——她说小芳，你昨晚和阿贤做了没有。

小芳头也没回——没有。

甜甜——真的假的，出发前最后一晚哎。

小芳拿了一瓶乌龙茶——他说要留到旅行第一天晚上。

甜甜吹了一声口哨——阿贤还挺浪漫的嘛。

小芳笑了一下——他有个屁浪漫。他是觉得旅行的时候做比较有新鲜感。

甜甜也拿了一瓶饮料。她拧开盖子喝了一口说——那他今晚可能要失望了。别忘了抽签。

小芳看了一眼窗外——阿贤正靠在车门上抽烟。他没有看她。

回到车上之后珍珍说还剩大概三个小时到。她说要不来玩个游戏。

阿伟说又在玩游戏。

珍珍说你可以不参加。然后开始讲规则。

游戏叫"不说真话就脱"。轮到你的时候你可以选择回答一个问题或者脱一件衣服。问题由全桌人投票决定最想问的一个。

佳佳第一个——他选了真心话。

珍珍想了一下——你现在坐在这里，最想操的人是谁——不是嘉嘉。

佳佳沉默了两秒。他转头看了一眼周围。然后他说——甜甜。

甜甜笑得很大声。阿伟的脸黑了一瞬但很快恢复了——他可能也不确定这是不是玩笑。

嘉嘉坐在佳佳旁边，面无表情地喝了一口水。有人说她生气了，但她开口说——他昨天跟我说过，所以我知道。

全场又安静了。

小芳第二个。她选了脱衣服。

她脱了一件薄外套。里面是一件白色的修身T恤，胸口的曲线被布料勾出来。她觉得所有人的目光都在她身上——不是错觉。阿贤看着她的眼神和平时不太一样了。

珍珍第三个。她选了真心话。

阿强问的——你最近一次自慰是什么时候。

珍珍看着他——昨晚。在想你的时候。

阿强的耳朵红了。

甜甜第四个。她选了真心话。

阿伟问的——你和佳佳上过床没有。

甜甜看着佳佳——没有。但我想过。

阿伟的表情没变，但他放在膝盖上的手握成了拳头。

游戏没玩完。因为前面的路拐进了一条土路，车开始颠簸，手机信号也开始断断续续。珍珍说快到了。


民宿比照片上好看。

是一栋白色的两层小楼——前面有一个院子，院子里铺着防腐木的地台，上面放着几把帆布椅和一张长桌。远处能看到草原的地平线。风很大，吹得晾在院子里的白色床单猎猎作响。

民宿老板是一对中年夫妻，女的胖胖的笑起来很和气，男的瘦高沉默，负责搬行李。老板娘说你们是毕业旅行吧，好好玩。她说你们的房间钥匙在这，自己拿。

桌上摆了十把钥匙，每把挂着一个木牌，上面刻着房间号。

珍珍看了一眼钥匙，然后掏出那叠纸条——现在公布今晚的抽签结果。

男生这边：
阿贤抽到和阿伟一屋——201。
阿强和佳佳一屋——202。
阿敏一个人一屋——203。

女生那边：
甜甜和小芳一屋——201。
珍珍和嘉嘉一屋——202。
静静一个人一屋——203。

等一下。
阿伟抬头——201是阿贤和我？那甜甜也住201？

珍珍说——对。三个人的房间是家庭房，有一张大床和一张单人床。

阿伟的脸色变了一下——那怎么分。

珍珍说——你们自己分。

甜甜已经拎着行李往楼上走了。她说我要大床。

小芳跟在后面没说话。

阿贤最后一个上楼。他从口袋里掏出那张纸条又看了一眼，然后折好塞回口袋。他没有跟任何人说那张纸条上写的其实是珍珍的名字——她在出发前偷偷把签换过。但那是另一个故事。


房间比想象中大。

201是一间朝南的家庭房。窗户很大，窗外能看到草原。一张一米八的白色大床摆在中间，靠墙的位置有一张行军床大小的单人床。地板是浅木色的，窗帘是亚麻色的。房间里有独立的卫生间，但门是磨砂玻璃的——能看到里面的人影轮廓。

甜甜一进门就把自己摔在了大床上。她说这床好软——她要睡这里。阿伟把行李放在墙角，站在窗边看外面，他没说话。

阿贤最后一个进来。他看了一眼床的布局——大床睡两个人没问题，但三个人就有问题了。但没有人主动提分配的事。

小芳在卫生间洗手。水声哗哗的。她从镜子里看到阿贤靠在门框上看她。她关了水龙头甩了甩手上的水——她说——你别这么看我。

阿贤——怎么看你。

小芳——就是那种看猎物的眼神。

阿贤笑了一下没说话。

甜甜在外面喊——你们俩在卫生间干嘛呢，快出来分床位。

小芳擦干手走出来。甜甜盘腿坐在大床中央，阿伟坐在床尾。阿贤靠在窗边。

甜甜先开口——大床我和阿伟睡。那张小的你俩谁睡。

小芳没说话。

阿贤说——我睡地板都行。

甜甜看了他一眼——得了吧你那么高睡地板明天腰都直不起来。这样吧。

她从床上跳下来——我和小芳睡大床，你和阿伟睡那张小床。

阿伟说那张床睡不了两个人。

甜甜说那就挤一挤嘛。你们俩大男人怕什么。

阿伟没再说话了。

小芳从行李箱里拿出睡衣去了卫生间。磨砂玻璃上印出她脱衣服的影子——先是上衣从头顶脱掉的轮廓，然后是牛仔裤从腿上褪下去的轮廓。甜甜吹了一声口哨。小芳在玻璃后面说了一句"别看了"。甜甜笑得更开心了。


晚上的风很大。窗帘被吹得鼓起来又塌下去。月光从窗帘缝隙里漏进来，在木地板上画了一道白色的线。

凌晨一点。

阿伟睡在小床靠墙的那一侧。阿贤睡在外面。小床确实太小了——两个人的肩膀碰在一起，阿贤能感觉到阿伟翻身的每一次动作。他在装睡，但他知道阿伟也没睡着。

大床上甜甜和小芳背对背躺着。甜甜的呼吸很均匀——不知道是真的睡着了还是在装。

然后阿贤感觉到一只手。

不是小芳的手——那只手从他的小腿开始往上摸。动作很轻——轻到如果不是在等它几乎感觉不到。手指沿着他的外侧脚踝往上，经过小腿肚，到膝盖侧面。

那只手不是小芳的。

阿贤没有动。他的呼吸保持平稳。那只手继续往上——摸到了他的大腿侧面。手指在他的大腿上停留了几秒，然后轻轻地捏了一下。

是甜甜的。小芳在大床的里面那侧——她够不到他。从方向来看，这只手是从大床边缘伸过来的。

阿贤在黑暗中睁开眼。他侧过头——甜甜面朝他侧躺着，月光只照到她的下巴。她的眼睛是睁着的。

她没有说话。她的手继续活动——从他的大腿滑到了他的裤子前面。手指隔着那层薄薄的棉布碰了他一下——他已经硬了。她用指尖沿着他的长度画了一道线。然后她缩回了手。

她翻了个身。

一切像没发生过一样。

阿贤躺在小床上，睁着眼睛看着天花板。旁边阿伟的呼吸不知道什么时候变得有点急促——也许他也醒着。也许他也感觉到了什么。也许这趟旅行从第一晚就不会有任何人睡得着。


凌晨三点，阿贤在走廊尽头碰见了珍珍。

他也说不清自己为什么在那个时间点走出了房间——可能因为那张小床太挤了，可能因为刚才那只手让他需要一点冷空气。走廊的灯是声控的，他走一步亮一截。

珍珍蹲在走廊尽头的自动售货机前面。她穿着一件长T恤当睡衣，下面露出两条光腿。她投了硬币，机器哐当一声掉下来一罐啤酒。她弯腰捡起来的时候回头看到了他。

珍珍——你也睡不着？

阿贤——床太挤了。

珍珍打开啤酒喝了一口，靠在墙上。月光从走廊尽头的窗户照进来，照在她的大腿外侧。她顺着他的目光低头看了自己的腿一眼，然后抬头看他。

珍珍——你刚才在看什么。

阿贤——没什么。

珍珍——你撒谎的时候会眨两次眼睛。你知道吗。

阿贤没说话。

珍珍又喝了一口啤酒，然后把罐子递给他。他接过去也喝了一口。啤酒已经不太凉了。

珍珍说——第一晚就有收获？

阿贤——什么意思。

珍珍——别装了。甜甜的手伸过来的时候我正好出来接水，看到了。

阿贤沉默了几秒——你看到了多少。

珍珍——够多了。

她把啤酒罐拿回去放在地上。然后她站起来——她的身高刚好到他的下巴。她凑近了一点，近到他能闻到她身上的沐浴露味——牛奶味的。

珍珍说——你知道吗，我换签了。

阿贤——我知道。

珍珍——那张签上本来是你的名字。我把小芳的签换给你了。

阿贤看着她——为什么。

珍珍——因为我想知道你是不是真的像她们说的那么厉害。

阿贤没有说话。珍珍伸手拍了一下他的胸口——晚安。然后她转身回了房间。走廊的灯在她身后灭了。

阿贤站在黑暗里。地上的半罐啤酒还在冒着微弱的气泡声。
