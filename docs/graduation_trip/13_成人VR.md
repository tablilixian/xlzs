# 毕业旅行 · 第六天下午 · VR

第六天。下午。乌兰察布市郊区。

珍珍在网上找到的一家VR体验馆——藏在一条背街的巷子里门面不大。招牌上写着"未来幻境"——四个字在LED灯箱里循环变色。门口贴着一张海报——海报上一个人戴着VR头盔脸上是惊讶的表情——广告语是"你想去的地方，VR都能到"。珍珍推开玻璃门的时候门上的风铃响了一声。

前台的女孩大概二十出头——穿了一身黑色的工装——她在刷手机。看到一群人进来她放下了手机——问几位。珍珍——十位。女孩——想要什么套餐——有单人体验和双人联机——还有沉浸式剧场。

珍珍——沉浸式剧场是什么。

女孩——你可以选一个场景。戴上头盔之后你会看到一个虚拟世界——里面的角色会和你互动。声音、触感、甚至气味——都有。

珍珍——触感是什么意思。

女孩——设备附带了体感服。你在虚拟世界里被碰到——身上的体感服会在对应位置产生压力反馈。所以你会感觉真的有人在摸你。

蒙古包里安静了。

珍珍——我们十个人。包场多少钱。

女孩报了一个数。珍珍没还价直接付了——微信到账的提示音响了一下。

女孩把他们带到了一个大的房间——大概有五十平方米。房间是空的——四面白墙——地面是灰色的软垫。墙角立着十套VR设备——头显、耳机、还有一件深灰色的紧身连体服。女孩说穿上体感服——等一下会有人来帮你们连接设备。

女孩出去了——门关上了。

十个面对十套紧身体感服。珍珍先动手的——她脱了自己的衣服套上了体感服。紧身拉链从胯部一直拉到领口——拉上之后整个身体被一层弹力面料包裹——每个人的身体曲线被勾勒得一清二楚。每个人都在穿——没有人说话。阿伟帮甜甜拉背后的拉链。嘉嘉自己反手拉上的。静静套上体感服的时候布料在她身上绷紧——她大腿内侧还残留着一小块青紫色的吻痕——从体感服的边缘露出来。

女孩带了一个技术员回来。技术员帮每个人调整头盔——检查体感服的传感器——确认触觉反馈的灵敏度。他手指按了一下珍珍的腰侧的传感器——她说有感觉——有点痒。技术员调低了那一个区域的灵敏度然后转到下一个人。

全部调试完毕。技术员——你们要什么场景。

珍珍——有什么选项。

技术员——海滩、森林、太空站、古堡、教室、还有——他的声音低了一点——成人内容需要单独授权。

珍珍——授权。

技术员看了她一眼然后递过来一个平板——屏幕上有一份免责声明。珍珍看都没看直接滑到底点了同意。她把平板传了一圈——每个人都在自己的名字后面点了同意。

技术员——你们十个人会被分配到同一个场景。场景里的所有角色——都由AI实时驱动。你们在虚拟世界里可以自由行动。碰到其他玩家——体感服会传递触觉反馈。每个玩家之间——也是。

他说每个玩家之间的触碰也会传递反馈的时候——阿强看了珍珍一眼。

技术员按了一个按钮。灯光熄灭了。头盔里亮起了一片白色。

他们站在一条街道上。

街道很真实——鹅卵石路面——两侧是中世纪的石头建筑——墙上有铁艺的路灯。天空是黄昏的颜色——橘色和紫色交织。空气中有海腥味——远处能看到海。

佳佳——我靠这太真了。

嘉嘉伸手碰了一下他的手——她碰到了——体感服在她手上产生了对应的压力反馈。她说——我碰到你了。

其他八个人分散在街道上。一个NPC从街角走来——穿了一身中世纪长袍——他走近的时候看着他们——用低沉的声音说——欢迎来到自由港——这里的夜晚什么都可以做。

NPC说完这句话就走过去了——消失在街道拐角。

珍珍——自由港是吧。

她沿着街道开始走。其他人跟着她。街道两侧有酒馆——有旅店——有一扇门半掩着的房间里面传出音乐声。珍珍推开了那扇门。

房间里灯光昏暗——中间有一张圆桌——圆桌周围坐着四个人——都是NPC——两个男人两个女人——穿着暴露的衣物。他们看到珍珍走进来——其中一个女人站起来——她绕着珍珍走了一圈——她的手指掠过珍珍的肩膀——体感服的反馈让珍珍的肩膀微微一紧。女人说——新人。

珍珍没有躲。她站在圆桌前——她的目光扫过这四个NPC——然后她回头看了一眼身后的其他人。她说——这场景有编辑模式吗——能不能把NPC换成我们自己人。

头盔里传来技术员的声音——可以。但需要权限。

珍珍——授权。

头盔里的世界闪烁了一下——然后圆桌周围的NPC消失了。取而代之的是空着的椅子。珍珍坐在了一把椅子上——她说——来——自由港——夜晚什么都可以做。

其他人坐下了。十个人围着一张虚拟的圆桌——在一个虚拟的中世纪酒馆里——穿着紧身体感服——每个人都能感觉到体感服下自己的心跳。

珍珍看着坐在她对面的阿贤——她说——你的手放在桌面上我能看到对吧。阿贤——对。珍珍——那如果我的手放在桌面下——你还能看到吗。

她把一只脚伸到了桌面上——在虚拟世界里她的腿出现在圆桌上——沿着桌面的黄铜色边缘——一直伸到阿贤面前的空气里。阿贤握住了她的脚踝。珍珍的体感服在脚踝位置产生了反馈——她感觉到了他的手——那是传感器模拟的热度和压力。

阿贤的手指沿着她的虚拟小腿滑了上去。

珍珍闭上了眼睛——她感觉到的是阿贤的手——还是体感服的传感器——还是她自己的想象——已经分不清了。

圆桌底下——更多的人在动作。甜甜的手放在阿伟的大腿上——她能感觉到他大腿肌肉的轮廓隔着体感服的布料。佳佳牵着嘉嘉的手指——两人的指尖在桌面下交叉。静静的手悬在桌面下——她不知道该放在哪里——然后一只手握住了她的——是阿敏。她僵了一瞬——然后放松了。

小芳坐在阿贤旁边——她能看到阿贤的手在珍珍的小腿上——她没有任何表情。她把手放到了桌面下——放在了阿贤的另一条腿上——不是大腿——是膝盖。阿贤看了她一眼。她说——继续你做你的。

技术员的声音从耳机中传来——检测到多组触觉交互。需要开启自由模式吗。

珍珍——开。

整个虚拟世界的灯光在一瞬间变成了深红色。

墙壁和地板上的纹路开始蠕动——圆桌的表面上浮现出了交错的图案——酒馆的门窗自动关上。墙壁变成了半透明——能看到外面的大海——海浪在夕阳下翻涌。

每个人身上的体感服的灵敏度在那一瞬间被调到了最高。

珍珍深吸了一口气——她说——现在谁都别装了。

VR头盔里的世界在继续。没有人按下退出键。
