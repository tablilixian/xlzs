# 毕业旅行 · 尾声 · 返程

第八天早上。太阳升起来的时候草原上的露水开始蒸发——整个草原笼罩在一层薄薄的雾气里——像地面上在冒烟。十个人在中午之前收拾好了东西——背包、纪念品、晾在院子里的泳衣——一件件塞回车里。民宿老板送了一箱当地的酸奶——珍珍没推辞。

退房的时候老板看着珍珍——他的表情像是在想说什么但又不知道怎么说——最后只说了——下次来提前打电话——我给你们留那间大通铺。珍珍——一定。

车的后备箱塞得比来的时候更满。十个人上车花了一点时间——因为每个人的位置和来的时候不一样。阿强坐在驾驶座——珍珍坐在副驾驶——她把座椅调后了一点把脚搭在中控台上。后排的座位分配在无声中完成——甜甜靠在阿伟身上——佳嘉和嘉嘉坐在中间排——小芳靠窗阿贤在她旁边——静静和阿敏坐在最后一排——两人之间隔着一个装满纪念品的袋子。

车驶出民宿大门的时候珍珍从后视镜里看了最后一眼——那排蒙古包在晨光中越来越小——直到拐过一个弯——从视线中完全消失了。

回程的途中车内播放着来的时候放的那些歌。佳佳在中间排睡着了——头歪在嘉嘉的肩膀上。阿敏靠着车窗戴着耳机不知道在听什么。静静在最后一排闭着眼睛——她的表情比来的时候少了些什么——比来的时候也多了一些什么。甜甜把车窗摇下来——风吹乱了她的头发。

中间在服务区停了一次。珍珍下车买了一包烟。她已经很久没抽烟了——她靠在车身上点了一根——吸了两口就掐灭了。阿强在旁边看着她——他说——回去之后——你住哪。珍珍——我租了一个房子在城南。离你公司不算远。阿强——你怎么知道我公司在哪。珍珍——我不知道——我猜的。

她又点了一根——这次抽了半根才掐掉。她弹烟灰的动作很用力——好像在想什么事。烟灰落在柏油路面上被风吹散了。

重新上车之后甜甜从前排探过头来——她说——回去之后谁也别装不认识我。佳佳——为什么要装认识你。甜甜踢了一下他的座椅靠背——睡你的觉。

嘉嘉——那个VR馆你记了电话没有——我想把那套体感服买下来。

珍珍——我有他们的微信。回去推给你。

阿强发动了车——九座车在高速上向北驶去。草原在车窗外倒退——变成丘陵——变成农田——变成村庄——城市的天际线在地平线上慢慢浮现。

当高速公路上的路牌出现了省会的字样——第一个红灯——第一个路口——车流从稀疏变得密集——整趟旅行在没有人宣布结束的情况下结束了。

珍珍在副驾驶上看着窗外不断后退的街景——商铺、公交站、骑着电动车穿行的外卖员——这一切和草原像是两个世界。她把手伸到车窗外面——风从她的指间流过。

阿强看了她一眼没有说什么。

在城市的高架桥上——九座车在午后的阳光中慢慢汇入车流——像一滴水落回了海里。

没有人知道这七天到底改变了什么。

但每个人都知道什么改变了。

珍珍收回手关上了车窗。她转头看了一眼后视镜——后座上的每个人都在做自己的事——玩手机、靠着窗户发呆、和旁边的人说话。她的目光在每个人脸上停了一拍——然后她靠回座椅里——她的目光投向前方。

前方是城市密集的楼群——高架桥在前方分叉——通往不同的方向。

她伸手调了一下空调出风口——冷风吹在她脸上。

好。她说。

阿强——好什么。

珍珍——没什么。就是好。

她伸手按了一下收音机的开关——一首老歌从扬声器里流出来——旋律在车厢里回荡。

没有人跟着唱。

但所有人都在听。

车轮碾过路面接缝处发出均匀的节奏声。车厢里很安静。风吹过车窗缝隙发出一种低沉的嗡嗡声——和来的时候一样。

九座车载着十个人和七天的行李——沿着高速公路——平稳地向前行驶。
