# 毕业旅行 · 第七天清晨 · 最后一抽

天亮前的那段时间——大概凌晨四点到五点半——是整趟旅行中最安静的。草原上的鸟还没开始叫。露水最重的时候——草叶上挂满了水珠——踩上去整只鞋都会湿。东边的天际线上有一层浅灰色的光带——太阳大概还有半小时才会露出来。

珍珍从大炕上坐起来的时候其他人还在睡。她看了一下周围——阿强蜷缩在靠近门的位置——他睡觉的时候眉头是皱着的。甜甜把脸埋在阿伟的腋窝里——阿伟的手臂环着她。佳佳和嘉嘉挤在一起——佳佳的一条腿搭在嘉嘉身上。小芳的头枕在阿贤的锁骨窝里——两个人在月光下看起来像一个人。静静和阿敏之间的空隙比睡着之前更宽了——两个人都睡在边缘——中间空出了一道足够再躺一个人的空间。

珍珍光脚踩在地面上——地面上凉凉的——她走到那扇小窗户前面——踮脚往外看。草原在晨曦中慢慢显露出轮廓——远处的蒙古包——风力发电机——电线杆——起得早的牧民从毡房里走出来。

她回头看了一眼。然后走到炕前——她的手指划过每个人的脚踝——轻轻地——一个一个——像点数一样。她碰到静静的时候静静睁开了眼睛。两个人对视了一秒——珍珍把食指放在嘴唇上——嘘。

静静又闭上了眼睛。

所有人起来之后珍珍说了一句话——她说今天不走。再住一晚。老板说可以续。

阿强——你不是说七天的行程吗。

珍珍——延一天。我付。

没有人反对。

第七天的白天在悠闲中度过。有人骑马——有人去镇上买了纪念品——有人就躺在院子里晒太阳什么都不做。珍珍去镇上买了一个新的转盘——比第一个更大——油漆更厚——格子更多——十六格。她回来的路上经过那家VR体验馆——前台女孩认出了她——她说你们昨天的体验有一个反馈——珍珍说——什么反馈。女孩说——系统记录显示你们那场自由模式的多人交互时长是四小时十七分钟——是我们开业以来最长的。

珍珍没说话。她把转盘夹在腋下走了。

晚上七点。最后一个蒙古包——最大那个。十六格的新转盘立在中间——所有人都到了。蜡烛点上了——一共六根——散放在蒙古包的各个角落。灯光不亮——刚好能看清转盘上的字。

珍珍说——今天是最后一天。昨晚我已经说过了最后一轮。但我想了想——昨晚那算中场休息。今晚才是真正的最后一轮。

她有新的规则。她说——每个人转一次——但每个人转的不是自己。你帮别人转。轮到你的时候——你指定一个人——你帮他转。转出来什么——他必须执行。然后换下一个人指定一个人。

佳佳——这不就是你帮别人安排活吗。

珍珍——对。所以你要想清楚——你给别人转到一个好的——还是你给别人转到一个难的——因为转完这一圈——下一轮就是那个人指定你。

第一轮——珍珍指定阿强。

她伸手拨动转盘。转盘旋转——十六格在烛光中轮转——停在了第8格——选一个人接吻到对方先停下来。

阿强环顾了一圈然后走到珍珍面前。他低头吻了她。吻了很久——久到蜡烛的火苗都跳了几下。珍珍没有推开他。但也没有回应。阿强先停下来了——按照规则——他停下了所以输了他喝了三杯酒。

第二轮——阿强指定佳佳。

他拨动转盘——第12格——被全场围观口交到射。

佳佳对着全场笑了笑——他脱了短裤——他的鸡巴已经半硬了。他躺在了毡垫上。嘉嘉作为他的女朋友跪到了他面前——但珍珍按住了她的肩膀——她说——规则没说是谁来做。被围观的是他——执行的人由指定的人决定。

阿强看了珍珍一眼然后说——阿贤来。

阿贤没有犹豫。他走到佳佳面前跪下来低头含住了佳佳的鸡巴。佳佳的头部后仰——他的呼吸变成了急促的喘息。阿贤的舌头和喉咙的技巧出乎所有人的预料——不到三分钟佳佳的身体就绷紧了——他的手抓住了毡垫的边缘——射在了阿贤的嘴里。阿贤站起来——他走到蒙古包角落把精液吐在了地上然后用啤酒漱了漱口。

第三轮——佳佳指定阿贤。

他转到了第6格——选一个人操——不能选自己的伴侣。

阿贤的目光掠过所有人。他站起来——走到静静面前——他伸出手。静静看了阿敏一眼——阿敏在喝酒——杯沿挡住了他的表情。静静握住了阿贤的手。

两个人面对面站着。阿贤没有带她去角落或暗处——他就在蒙古包中央——在烛光的包围中——把她放倒在毡垫上。他掀起了她的裙子——她的内裤是一条很普通的纯棉白色款——在烛光下几乎是半透明的。他没有脱她的内裤——他隔着那层棉布用嘴唇贴上了她的阴部。静静在那一瞬间全身软了。

他的舌头隔着棉布在她的缝隙上来回滑动——她的体液很快就浸透了那块布料。他用牙齿叼住内裤的边缘拉到了一边——她的阴唇从布料下滑出来——他直接含了上去。她发出了一声破碎的呻吟。阿敏把酒杯放下了——杯底在桌面上放重了。

阿贤用舌头操了她大概五分钟——直到她的身体开始发抖——直到她的手指在他头发里揪紧——直到她高潮时他的下巴上全是她的水。

他站起来拉好裤子。静静躺在垫子上用一只手臂遮着眼睛——胸口起伏着。

第四轮——阿贤指定小芳。

他转到了第3格——全裸跳舞三分钟。

小芳当着所有人的面把衣服脱光——一丝不挂——然后她跳了一支舞。没有音乐——她自己在烛光中摆动身体——她的乳房在跳动中晃动——她的腰肢像一条蛇。她跳舞的时候看着阿贤——只有他。他也在看她——但他没有笑。

第五轮——小芳指定嘉嘉。

她转到了第14格——被绑起来——让全场每个人在你身上留下一个印记——不能是同一个位置。

佳佳用珍珍带来的绳子把嘉嘉的手腕松松地绑在了蒙古包的木柱上。每个人依次走到她面前。阿强在她的锁骨上留下了一个吻痕。阿伟在她的腰侧用指甲画了一道轻微的红痕——会持续到明天的。甜甜在她的左乳头周围留下了两排齿印——不深——刚好会痛到明天。静静在她的膝盖窝里留下了一个吻——最轻的触碰。阿敏在她的右手腕内侧轻轻咬了一下。阿贤在她的尾骨位置——她的体感服最敏感的位置——用拇指按压了十秒。佳佳——在嘉嘉的额头上落了一个吻——在所有痕迹中最不情色但最温柔的一个。珍珍站在她面前——她低头在嘉嘉的颈窝里留下了一个湿热的吻——持续五秒——用嘴唇感觉到了她的脉搏。小芳——在嘉嘉的大腿内侧——用舌头画了一个完整的圆。

每个人都在她身上留下了印记。佳佳把她的手解下来的时候她身上有了十个人的痕迹。

第六轮——嘉嘉指定阿伟。

她拨动转盘——第5格——蒙眼让人认出每一根触摸你的手指。

所有人都蒙上了自己的眼睛——阿伟也是。十个人围成一圈坐着——阿伟坐在圈中央。珍珍说——你的手放在身前——每个人会碰一下你的手——你要说出是谁。所有人一起伸手——十只手同时落到阿伟的手臂上——手指的区别——粗细长短——温度——茧的位置——他不需要看。他说——左数第二根——佳佳。第三根——阿贤。第五根——静静。第八根——嘉嘉。全部对了。

第七轮——阿伟指定甜甜。

转到了第10格——再转一次。

他转了一次——第2格——脱两件衣服。甜甜脱了上衣和短裤然后坐了下来。

第八轮——甜甜指定阿敏。

转到了第16格——自己写一条规则——所有人都要遵守。

阿敏站起来——他走到静静面前对她说——大二那年冬天滑雪场的事——对不起。这是我写的规则——我道歉。你可以不接受——但这是我的规则。

蒙古包里安静了大概五秒。静静伸出手——她的手指碰到他的脸颊。但她什么也没说——她把手指收回来了。

第九轮——阿敏指定静静。

转到了第7格——被选中的两个人——交换今晚的床伴。阿敏写道——我和阿贤交换。

第十轮——静静指定了阿贤。她转了——第15格——金色的格子——免死金牌——可以跳过——也可以指定别人跳过。静静选了跳过她自己——但她把免死给了阿敏。

转盘还在继续。夜越来越深。蜡烛一根接一根地烧完——珍珍换上了新的。蒙古包的地面上洒了酒——毡垫上有各种污渍——空气里混着汗味酒味和性器的味道。十六格转盘上还有未转到的格子——第13格——坦白——说出一个这七天里你做过但没人知道的事。第9格——选一个人陪你睡到天亮——纯睡觉。

转盘又转了两轮——有人哭了——没有人在意——有人接吻了——没有人在意——有人射了——有人高潮了——没有人在意。

最后一轮——转回到了珍珍。指定的人是她自己——她拨动了转盘。转盘旋转了很久——像一个永不停歇的循环——终于慢慢停下——停在最后一格——第1格。

第1格上写着——重新开始。

珍珍看着那个字——蜡烛的火苗在她眼睛里跳动。她说——什么意思。没有人知道——因为这个格子是空的——她买转盘的时候第十六格是空白的——她自己用油漆写上了重新开始四个字。

她看着那四个字看了很久。然后把整瓶威士忌端起来——仰头喝完了一半。

她说——天快亮了。

她站起来走到蒙古包门口——掀开门帘。晨曦从门帘的缝隙中漏进来——灰蓝色的光——带着草原清晨特有的清冷。

她回头看了一眼——其他人还在烛光里。身影被拉长投在毡壁上。

她走出去——赤脚踩在沾满露水的草地上。冷意从脚底穿上来。

阿强跟着她走出来——站在她旁边。两个人站在晨光中。远处有牧民已经开始了一天的劳作——毡房顶上冒出了炊烟。珍珍说——你觉得会有第二场旅行吗。阿强——会。珍珍——你怎么知道。阿强——因为转盘上写了重新开始。

珍珍没有说话。晨光中她的嘴角弯了一下——很轻——不太确定是不是笑了。

蒙古包里传出了佳佳的笑声——然后是嘉嘉在骂他——然后是大家的笑声——混在一起——

像这七天里的所有声音叠在了一起。
