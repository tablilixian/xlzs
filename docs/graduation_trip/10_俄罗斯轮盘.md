# 毕业旅行 · 第五天 · 俄罗斯轮盘

第五天中午。雨。

草原上的雨来得没有预兆——早上还是晴天中午突然乌云压境。雨点砸在蒙古包的顶棚上像有人在上面倒豆子。空气里的温度降了至少十度。珍珍把所有人叫到大蒙古包里——最大的那个蒙古包直径有八米，里面铺了一圈羊毛毡垫子，中间有一个铁皮炉子但今天没生火。光线从顶部的天窗漏进来——灰白色的天光——把每个人的脸照得像在暗房里。珍珍拿出了一个六发左轮手枪形状的打火机——她在指尖转了一圈然后放在毡垫上。她说今天不转盘了。今天换一个。

她从背包里拿出一副扑克牌——抽出了五张红心和五张黑桃——一共十张。

规则很简单：每人抽一张。抽到红心的人享受。抽到黑桃的人服务。然后抽签组合——数字相同的配对。红心2和黑桃2做一组。红心3和黑桃3。红心4和黑桃4。红心5和红桃5。红心6和黑桃6。但数字越小的组先开始——2最先。然后依次进行——后一组可以旁观前一组。没有人可以离开蒙古包。直到全部完成。

佳佳——你这又是凌晨几点做的规则。珍珍——凌晨四点。我昨晚只睡了两个小时。来坐下。

十个人围成一圈坐在毡垫上。珍珍把十张牌洗了三遍然后摊开在垫子中间。她先抽——她翻开——红心5。阿强抽——黑桃5。他看了珍珍一眼把牌放进口袋。

其他人依次抽牌。甜甜——红心3。阿伟——黑桃3。嘉嘉——红心2。静静——黑桃4。阿贤——红心6。小芳——黑桃6。阿敏——红心4。佳佳——黑桃2。

数字2的先来。

嘉嘉是红心2——她享受。佳佳是黑桃2——他服务。所有人都看着。嘉嘉脱了自己的T恤和短裤躺在了毡垫上——她只穿了一条白色的蕾丝内裤。佳佳跪在她两腿之间。他俯下身——从一个吻开始。他的嘴唇贴着她的小腹——沿着腹中线往下一直吻到内裤的边缘。他把鼻子埋在那块白色的布料上深深吸了一口气——他的呼吸引得嘉嘉的腹部收缩了一下。他用牙齿咬住了内裤的边缘——她抬起臀部让他脱了下来。毛不多的阴户暴露在灰白色的天光下。

佳佳的脸埋了进去。没有前戏铺垫——直接是唇舌并用。他的舌头从下往上沿着她阴唇的缝隙一路舔到阴蒂然后用嘴唇含住了它。嘉嘉发出了一声很长的呼吸——不是呻吟——是比较用力的吐气。她的手指插进了他的头发里——她闭着眼睛。

其他八个人在旁边看着。没有人说话。

佳佳的舌头围绕着阴蒂画圈——然后在湿透的阴唇间上下抽动模拟着性交的动作。他的鼻尖陷在她的阴毛里。他的一根手指从下方进入了她——嘉嘉的身体向上弓了一下——他加入了第二根手指。他的舌头在她阴蒂上快速振动——同时手指在她的阴道里弯成了钩状按压着前壁。嘉嘉的腿开始颤抖——她的脚趾蜷缩——她的臀部微微抬起——她高潮了。她没有发出很大的声音——喉咙里只有一声极短的气音——但她的身体在一瞬间完全僵直然后软了下去。

佳佳抬起头——他的下巴湿透了。嘉嘉躺在毡垫上看着天窗——她花了大概十秒才坐起来穿衣服。

数字3。甜甜红心——阿伟黑桃。

甜甜侧躺在了毡垫上。阿伟从背后贴上去。他没有脱她的衣服——直接把她的裙子推到了腰上。她穿了一条很薄的三角内裤——他看到了她的臀缝中间那层布料已经湿了。他把内裤拉到膝弯。他扶着已经硬得发紫的鸡巴——龟头在她湿透的阴唇之间上下滑动了两下然后插了进去。甜甜发出了一声被压低的啊——他的手指绕到前面按住了她的阴蒂。

阿伟抽插的速度不快但很深——每一插都顶到最深处才抽出来。甜甜的嘴唇微微张开——她睁着眼睛看着前方的虚空。她的手指在地面上无意识地抓着垫子的边缘。阿伟一只手按着她的髋骨调整角度——另一只手绕到前面继续揉着她的阴蒂。他的呼吸在她耳朵后面越来越重。甜甜说了一句什么——声音太小没有人听清——然后她的身体弓了起来——高潮来的比预期更快——她夹紧了他的鸡巴然后松开了。

阿伟没有射。他拔出来——龟头上沾满了透明的液体在灰白色的光线下发亮。他拉上了短裤。

数字4。静静红心——阿敏黑桃。

静静和阿敏是真正的情侣。在场的人都以为这轮会最平淡——但他们错了。静静主动脱了衣服——全身赤裸躺在垫子上。阿敏没有脱——他只是拉开了拉链。他跪到她面前——鸡巴硬着但表情很冷。他没有前戏。他直接压了上去——插进去的时候没有任何温柔。静静叫了一声——皱着眉但没让他停。阿敏的动作像一个想证明什么的人——又急又重。他根本不管静静有没有湿。她干涩的阴道被强行撑开——每一次插入都伴随着疼痛的表情但她在忍。阿敏把她的腿压到胸口的位置——更深地进入她。静静地眼睛盯着蒙古包的天窗——眼泪从眼角滑进头发里但她没有出声。阿敏射在她里面。他拔出来的时候她的阴道口泛红——有一些白色混合着血丝的液体从中流出来。

没有人说话。静静把腿放下来躺着——过了一会儿才坐起来。

数字5。珍珍红心——阿强黑桃。

珍珍和坐在对面的阿贤对视了大约两秒——然后看向阿强。阿强放下他一直在转的免死金牌。他没有走过来。他坐在原地——说了一句让所有人都意外的话。他说——我不服务。我转免死。

他从口袋里掏出金牌放在垫子中间。

按照规则——免死金牌可以跳过一轮。蒙古包里安静了大概五秒。珍珍看着那枚金牌没有说话。阿强的目光和珍珍的目光在空中交汇。阿强先移开了视线——但他没有收回免死金牌。珍珍——那你跳过了谁来服务。

阿强——谁拿到5的黑桃谁来服务。

那双目光重新交汇。珍珍低头看了一眼自己面前的扑克牌——红心5。她说——黑桃5在谁手里。

阿贤从口袋里掏出了一张牌——黑桃5。

原来他抽到的就是黑桃5。阿强抽到黑桃5之后——在所有人看牌之前——和阿贤换了。

珍珍看着那两张交换过的牌——沉默了几秒然后笑了一声。她站起来脱了裙子。她躺下的时候说——你来。

阿贤走过来跪在她两腿之间。他的手指沿着她的胫骨往上滑到膝盖然后大腿内侧——他低头亲了一下她的髋骨左侧。珍珍闭上了眼睛。

阿贤用嘴唇和手指侍弄了她很久——久到旁边有人开始坐不住了。他两根手指在她穴里慢慢进出——掌心压着她的阴蒂——每一次旋转和按压都精准地落在她最敏感的位置。珍珍的呼吸越来越浅——她的腰开始不自觉地跟着他的手腕起伏。她高潮的时候没有掩饰——喉咙里发出一声拖长了的呻吟——湿透的液体顺着他的手指流到了垫子上。

她躺了一会儿睁开眼睛看着蒙古包的天花板。她说——阿强。阿强——嗯。她说——谢谢你。阿强——不客气。

数字6。阿贤红心——小芳黑桃。但阿贤的黑桃5刚刚已经用了。规则乱了一瞬——每个人都在看珍珍。珍珍没有说话——她刚高潮完还在调整呼吸。甜甜接过了规则——最大的数字不需要抽了。谁拿到6谁走最后一轮。

小芳站了起来——她是黑桃6。

选择享受的人不在——阿贤的黑桃5已经用了。甜甜看着小芳——你可以选一个还没用过的红心。或者——你选任何人。小芳的目光在所有人脸上停了一圈然后停在了静静身上。静静还半裸着腿上流着阿敏的精液。小芳走到她面前——她蹲下来用拇指擦了一下静静大腿内侧那道混着血丝的白色。她把拇指含进了嘴里。

然后她低头舔了上去。

她的舌头从静静的大腿内侧开始——一路往上——把流出来的精液和血丝舔干净。静静的身体颤抖了一下。小芳的嘴唇覆上了静静还肿着的阴唇——她的舌头温柔地拨开大小阴唇——舌尖探入阴道中把深处的液体勾了出来。静静的手按在地面上——指节发白。小芳的舌头在外面画圈——她的嘴唇含住了阴蒂轻轻吸吮。静静的身体在一瞬间僵住了——她发出了一个所有人都没听过的声音——不是呻吟不是哭泣——是一种混合着疼痛和释放的破碎的吐气。她的高潮来得很剧烈——小芳没有停下来——她用舌头的快速振动把她推进了第二次高潮。

静静的身体瘫软在垫子上——她的眼前有光斑在浮动。

小芳站起来擦了擦下巴。她的嘴唇上沾着混合的液体——阿敏的精液和静静的体液和自己唾液。

她走回自己的位置坐下。外面的雨还在下。

十个人坐在蒙古包里。空气里弥漫着精液和体液的气味混合着潮湿的羊毛味。没有人再说话。垫子上有几块深色的湿痕——不知道是谁的。

天窗漏进来的光更暗了——云层更厚了。雨不知道什么时候会停。

珍珍躺在垫子上还没有起来——她看着天窗——雨水打在天窗的玻璃上沿着弧线滑落。她说——晚上还有一轮。
