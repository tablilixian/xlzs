# 毕业旅行 · 第五天深夜 · 走错的门

俄罗斯轮盘结束后每个人都洗了澡。热水冲了很久。有人换了一身干衣服有人裹着浴巾就回房间了。雨在傍晚的时候停了——草原被洗过一遍空气里有泥土和草根的味道。天晴了但云层还没散尽——月光从云的缝隙里漏下来——院子里的一切都在明暗之间反复变化。

民宿的走廊在晚上十一点之后只剩一盏昏黄色的地灯。走廊铺了深色的地毯——光脚踩上去没有声音。门和门之间的间隔不到两米——每一扇门看起来都差不多——同样的深色木门同样的黄铜把手。

珍珍的房间在最东边。阿强的房间在她隔壁。再过去是佳佳和嘉嘉。然后是甜甜和阿伟。静静和阿敏。小芳和阿贤。走廊尽头是一扇窗户——窗户打开着——夜风把窗帘吹得鼓起来像一个鼓满的帆。

住在民宿的人大概都知道走廊尽头那扇窗户在夜里十一二点左右会有夜风吹进来带起草坪上泥土的味道。但今晚知道这件事的不只民宿的人。

事发前的铺垫发生在晚餐时间。

晚餐是民宿老板做的烤羊排——配了土豆和洋葱。十个人围坐在长桌前。佳佳讲了一个笑话——关于一个俄罗斯轮盘的——没有人笑。佳佳说好吧今天笑话不太适时宜。珍珍切了一块羊排放进嘴里嚼了很久。阿强坐在她对面——中间隔了一整张桌子的蜡烛和餐具。他的目光在她脸上停了一会儿然后移开了。他的手机屏幕亮了——他低头看了一眼然后翻了过去。

晚餐散得比平时早——大概九点半就没人再动刀叉了。

阿贤第一个回房间——他说今天累了。小芳没有跟他一起走——她留在餐桌上和静静说话。静静的状态从俄罗斯轮盘之后就不太对——她说话的时候声音比平时轻——笑的时候嘴角的幅度很浅。小芳给自己倒了一杯红酒然后给静静也倒了一杯——她推过去的时候手指在杯沿上停了一下。

十点半。珍珍回房间。她穿过走廊的时候经过了那扇窗户——窗帘鼓起来的时候带起的风拂过她的小腿。她的房间门锁有一点问题——要把钥匙往里推一下再转才能打开——她的手在锁上忙了几秒才搞定。

十一点。佳佳和嘉嘉从浴室回来——裹着两条浴巾经过走廊的时候水珠从他们的头发上滴在地毯上留下深色的圆点。他们的房间门关上了。灯亮了一会儿然后调暗了。

十一点一刻。走廊里有人。

不是从房间出来的——是本来就在走廊里的人。那个人站在走廊中间——在昏黄色的地灯光线下——面朝着走廊尽头的窗户。窗户的窗帘在夜风中鼓起来又落下——投在墙上的影子像某种生物在呼吸。

那个人开始走。每一步踩在地毯上都没有声音。

走到第一扇门前停了一下——那扇门里的灯已经全黑了。没有停留太久——继续走。第二扇门——灯还亮着——门下缝隙漏出一条暖黄色的光。没有停。第三扇门——灯关了。门缝下面漆黑一片。

那个人在第三扇门前站住了。

关于这扇门里的情况：这扇门里住的是甜甜和阿伟。甜甜十点半就睡了——阿伟还醒着——他在床上刷手机——关着灯但他还没有睡意。他听到门外有脚步声在门口停住了。他看着门缝下方的地面——没有人影——门缝很窄——只能看到模糊的光影变化。他以为是住客经过就没有在意。

门把手动了一下。

没有被推开——只是被握住了扭了一下——没有扭到底——像是有人在确认这扇门锁了没有。阿伟从床上坐起来——他说——谁。

门外没有回答。

脚步声往走廊另一端去了——很轻——被地毯吸收了。

阿伟掀开被子走到门口——他拉开门探头看了一眼——走廊里空无一人。他关上门重新锁好。

但在他关门之前的光景——在他拉开门的那一瞬间——走廊尽头的窗帘被风吹起——月光短暂地照亮了走廊的一半。他看到了走廊尽头的拐角处——一个身影闪进了那个方向——唯一通往的地方是走廊尽头的窗户——以及窗户旁边的一个凹位——放了一盆半人高的绿植。

阿伟没追过去。他关上了门。

但甜甜在那时候醒了——她说——外面有人。阿伟——没人——可能是风。

甜甜——不是风。有人开了我的门。

阿伟——没有——他只是碰了一下。

甜甜——你知道是谁。

阿伟——没有看到。

甜甜沉默了。她翻了个身背对着他。

同一时间——走廊那一端——珍珍的房间。有人站在珍珍的门前。

这扇门的锁有问题——需要把钥匙往里推一下再转才能锁好。如果关门的时候只是轻轻带上——锁舌不会完全弹出来——门会处于一种"关了但没锁"的状态。珍珍今天洗澡之前从房间出来——经过走廊去拿充电器的时候——关门的动作可能轻了一些。

那双手握住了这扇门的把手。

往下压——没有阻力——门开了大概三厘米的缝。

那双手停了一下。然后推开了门。

房间里漆黑一片。窗帘拉着——月光也漏不进来。床上有人——被子中间隆起的轮廓——呼吸平稳——应该是睡熟了。

那人进了门。门在身后关上了——咔嗒一声——这一次锁上了。

脚步声在门口停留了几秒——让人适应黑暗——然后开始向床边移动。

床上的被子动了一下——睡着的人翻了个身——翻到了背对门的一侧。一条手臂从被子里伸出来搭在枕头上。

那人站在床边——站了很久——大概有一分钟——然后坐下了——坐在床沿上。床垫因为增加的重量微微凹陷——弹簧发出一声极轻的吱呀。

那人伸出了手——手指碰到了搭在枕头上的那条手臂——从手腕开始——沿着前臂——往上——一直摸到肩膀。皮肤是温热的——睡眠中的温度比清醒时高一些。

被子下面的人发出了一声含混的哼声——像是在半睡半醒中被人触碰的疑问。

那人没有回答。手指从肩膀滑到了后颈——那里的皮肤最薄——指尖能感觉到脉搏的跳动。

被子下面的人动了一下——翻回身来。

在那双眼睛看清楚是谁之前——在被子掀开之前——在任何人出声之前——

走廊里突然传来了另一阵脚步声。从远处来的——从走廊另一头——朝着这个方向——越来越近——在那扇门前停住了。

然后是敲门声。笃笃笃。三下。间隔均匀。

然后是阿强的声音——珍珍——你睡了吗。

床沿上坐着的那个人没有动。

床上的人没有回答。

敲门声又响了——还是三下——间隔和第一次一样——很均匀——不急促——像一个人给了另一个人足够的时间来决定要不要应答。

珍珍的声音从被子里传出来——她说——睡了。什么事。

阿强在门外——没什么——就想问你明天早餐想吃什么都行。

珍珍——知道了。晚安。

阿强——晚安。

脚步声从门外离开了——沿着走廊回到了隔壁那扇门——关门声——锁舌弹进锁扣的声音——安静了。

床沿上坐着的那个人在黑暗中仍然没有动。又过了大约十秒——那个人站起来——无声地走到门口——轻轻拉开门——侧身出去——门在身后再次关上——这一次没有锁上——和来之前一样。

走廊里空无一人。地灯还是那盏昏黄色的灯。窗帘还在窗口鼓动。

没有人知道那个坐在床沿上的人是谁。

但甜甜第二天早上吃早餐的时候说了一句话——她问珍珍——你房间的锁修好了没。珍珍——没修——怎么了。

甜甜——昨晚有人开了我的门。我以为是风。

珍珍端起咖啡喝了一口——没接话。

阿伟看着她们两个——他把手里的烤面包放下了——但他什么也没说。
