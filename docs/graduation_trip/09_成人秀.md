# 毕业旅行 · 第四天夜晚 · 成人秀

县城边缘有一家夜总会——招牌不大，门口停的车不少。珍珍说今晚不去不合适——既然来了草原怎么能不看一场草原特色的表演。没有人问她怎么知道这个地方的。她提前订了座——十个人坐在靠前的圆桌，桌上放了一瓶伏特加和一大桶冰块。灯光暗蓝色的，座位的椅套是红色丝绒的。

舞台不大，离地面大概半米高，三面围了观众，第四面是一块深紫色的幕布。主持人穿了一身亮片西装——站在幕布旁边说欢迎各位来到草原之夜。表演已经开始——开场是一个钢管舞女郎，身材纤细，绕着钢管旋转的时候身体像没有骨头一样。

但当主持人宣布今晚有观众互动环节的时候，珍珍端起了酒杯。

主持人：接下来这个节目——需要一位勇敢的先生上台配合。

佳佳站了起来。他站起来的时候嘉嘉拉了他一下——她说你干嘛。佳佳——玩游戏。

他穿过观众席走上舞台。主持人给他戴了一个黑色眼罩——然后用绳子把他的手腕松松地绑在身后——说这种绳子一挣就开。佳佳站在台上——穿了一件白色T恤和黑色短裤。舞台灯光打在他身上。

幕布打开了。台上放了一把椅子。主持人让他坐下。他坐了。主持人说——先生，等一下会有人上来跟你做一些游戏。你的任务是——猜出是谁。每猜对一次——奖励。猜错一次——惩罚。

台下有人起哄。主持人拍了拍手。第一个人走上台。

那是一个女人穿着高跟鞋。她走到佳佳面前——用手背碰了一下他的脸颊。然后她的手指沿着他的下颚线滑到下巴——然后滑到了他的胸口——手掌贴着他的T恤——她的手指沿着他胸肌的轮廓画了一个圈。然后她低下头——隔着T恤含住了他的乳头。佳佳的身体在那一瞬间绷紧了。他听到了周围观众的笑声和起哄声。她的舌头绕着那粒凸起打转——用牙齿轻轻咬了一下布料的边缘然后松开了。她站直了身体。

主持人——猜。是谁。

佳佳——嘉嘉。

台下的观众发出一片混杂的声音——有人鼓掌有人笑。主持人——确定吗。

佳佳——确定。

主持人——错了。惩罚。

那个女人摘下了假发——是珍珍。

她回到座位端起酒杯冲佳佳笑了一下。佳佳在台上听到她的声音才知道是她——她说——你猜你女朋友会第一个上吗。第一轮永远是别人。

第二轮。第二个人走上台。

这次的脚步声更轻。她走到佳佳面前——没有碰他的脸——直接跪在了他两腿之间的地上。她的手放上了他的膝盖——沿着大腿内侧往上滑——滑到了他短裤的拉链位置。她的手指拉下了拉链——隔着内裤握住了他已经硬了的鸡巴。佳佳的头部后仰碰到了椅背。

她隔着内裤用嘴含住了他的龟头——她呼吸时的热气透过那层薄薄的棉布传递到他的皮肤上。她的嘴唇沿着龟头的形状——隔着布料——描绘了一圈。佳佳的呼吸变得不均匀了。

主持人——猜。

佳佳——……甜甜。

错了。惩罚。

台上的女人站起来——她把及腰的长发从面前拨开——是静静。静静回到座位端起茶杯喝了一口。阿敏在看佳佳的表情——佳佳的眼皮在眼罩下动了一下。

第三轮。第三个人走上台。

脚步声几乎听不到——光脚。那个人走到佳佳面前停住了。佳佳等待着——什么也没有发生。大概过了十秒——他以为人已经走了——然后他感觉到了一条温热的东西碰到了他的嘴唇。是一个人的舌头——从下方往上——沿着他嘴唇的缝隙——像猫舔牛奶一样——轻轻舔了一下他的上唇然后离开了。

佳佳的嘴唇微微张开。那条舌头没有再回来。

主持人——猜。

佳佳——……小芳。

主持人——对了。奖励。

台下有人吹了一声口哨。佳佳的奖励是他的眼罩被摘掉了——他眨了两下眼睛适应灯光——看到小芳站在他面前穿着一条和上台前一样的牛仔裤和白T恤。她说——你猜得挺准的。佳佳——你刚才碰我嘴唇的方式和阿贤亲你的时候一样。

小芳笑了笑回到座位。阿贤没有说什么——他的手在桌布下放在自己腿上紧握了一下。

主持人说今晚的互动环节到此结束。观众鼓掌。佳佳从台上走下来的时候嘉嘉握住了他的手——他的手指是冰凉的。

第四轮不是表演的一部分。发生在散场之后。

从夜总会出来的时候草原的风很大——吹得人站不稳。十个人走回停车场——停车场只有一辆出租车亮着顶灯和他们的九座车。路灯间隔很远之间有大量黑暗的空隙。珍珍走在最前面阿强跟在她后面。阿贤和甜甜走在队伍的最后面——没有人回头看过一次他们。

在夜总会建筑侧面的阴影里——从停车场看不到的一个死角——甜甜把阿贤推到了墙上。她说——刚才台上你看到静静舔他的时候硬了没有。阿贤没有回答。甜甜说——你硬了。

她用膝盖顶了一下他的裤裆——隔着裤子感觉到了那根鸡巴的硬度。

她说——跟我来。

两个人没有直接回到车上。花了大概十五分钟。等他们回到停车场的时候其他人已经在车里等了一阵了。甜甜拉开侧门坐进去的时候珍珍从驾驶座回头看了她一眼——甜甜的头发比进夜总会之前更乱了一些——嘴唇上有一条新的口红的痕迹——不是她自己的颜色。

珍珍转回去发动了车。没有人问为什么这么久。

车驶出停车场的时候远光灯照亮了草原公路前方的一小段路——两侧的黑暗无限延伸。车里很安静——只有发动机的声音和风声。

副驾驶的阿强看了后视镜一眼然后转回去看着前方路面。

他从口袋里掏出那枚金色的免死金牌在手里转了一下然后放回了口袋。
