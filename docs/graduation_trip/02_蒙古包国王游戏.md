# 毕业旅行 · 第二天 · 蒙古包国王游戏

第二天早上九点，车开了四十分钟进入草原深处。碧绿色的大地一直铺到天边。风很大，吹得草一层一层倒下去又站起来。营地的蒙古包散落在一片缓坡上——五六个白色的圆顶建筑，远处有一条小河，河边的马在低头吃草。

蒙古包比想象中大。直径大概七八米，中间放了一张矮桌，周围铺了一圈厚厚的羊毛坐垫。靠窗的位置堆着几卷备用的毯子。窗户很大——正对着外面的草原，窗边垂着几幅白色的纱帐，平时拢在两侧用绳子系着。纱帐很薄——透光，薄到你能看到纱帐后面窗框的轮廓。蒙古包的中央生了一个铁炉子，烟囱从包顶伸出去，炉火烧得正旺。

营地的老板说今晚外面会放露天电影——幕布就挂在蒙古包正前方的两根木桩之间。从蒙古包的窗户看出去正好能看到幕布的背面。

珍珍把带来的酒和零食全部摆在矮桌上。白酒、啤酒、马奶酒一字排开。烤羊被端上来的时候还冒着油星子——表皮烤得焦脆，里面的肉嫩得流汁。大家盘腿围坐在矮桌四周。

几个小时后，酒已经下去了大半瓶。蒙古包里暖得像夏天。炉火烧得噼啪响。外面天已经黑了。电影开始了——是一部草原题材的纪录片，画面在幕布上缓缓流动。光线从窗户透进来，把纱帐染成流动的颜色。

珍珍端着酒杯站起来——她说我们不玩国王游戏了。太慢了。

她从手机上调出一个随机数字生成器——每轮生成一个数字，指定一个号码和一项任务。号码由她喊停决定。没人知道下一个被叫到的是谁。

她从口袋里掏出一把分好号码的纸条——每个人抽一张。

第一轮。珍珍喊停。屏幕上停住的是：3号和7号——用纱帐围起来，在里面互相抚摸三分钟。

全场安静了一拍。纱帐——有人看了一眼窗边那几幅薄薄的白色织物。

3号是阿强。7号是嘉嘉。

两个人站起来走到窗边。他们把两边的纱帐拉开在窗户前面围出了一个半圆的小空间——纱帐拢合在一起，形成了一个视线遮挡的临时隔间。从外面看，纱帐里面只有模糊的影子轮廓。很保险——很安全。

嘉嘉先开口——三分钟。你打算怎么摸。

阿强站在她面前——纱帐里的空间比他想象中小。她的脸离他很近，近到能闻到混着马奶酒味的呼吸。

他说——你想怎么摸。

嘉嘉没有说话。她伸手解开了阿强衬衫最上面的两颗扣子，把手指伸进去——指尖贴着他的锁骨慢慢滑到胸口。她的手指在他的胸肌上画了一个圈，指甲轻轻刮过他的乳头。阿强的呼吸变沉了。她低头隔着衬衫咬了一口他的乳头——他吸了一口气。她的手一路往下滑——隔着裤子握住了他已经鼓起来的鸡巴。她隔着布料揉了一把评估了一下尺寸——然后松开了。

她的手从他裤子里抽出来放在他的腰上。两个人隔着衣服贴在一起站着——她的大腿根部贴着他的裆部。她动了动腰，隔着两层布蹭了蹭他已经硬了的鸡巴——阿强的喉咙里发出一声低哑的闷哼。

嘉嘉嘴唇凑到他耳边——你硬得挺快的。阿强没说话，他的耳朵红透了。

三分钟到了。纱帐拉开的时候阿强的耳根是红的。嘉嘉的表情没什么变化——但她的嘴唇比进去之前红了一点。只有一点。

外面电影正在放草原的日落——整个画面是金黄色的。光线从纱帐后面打过来的时候有人注意到一个细节——纱帐上的影子比想象中清晰。清晰到能看出两个人站的位置之间的距离。

注意到这个细节的人是佳佳。他什么也没说。


第二轮。屏幕上停住：5号和2号——纱帐，互相脱衣服，直到其中一方全裸。

5号是佳佳。2号是甜甜。

两个人站起来的时候阿伟的表情变了——但甜甜没有看他。她径直走向纱帐。纱帐在她身后合拢。

电影画面正在一片广袤的草原上移动——草是绿色的，天是蓝色的，光线很充足。纱帐上清晰地映出两个人的轮廓。佳佳站在甜甜面前，甜甜背对着窗户站着。

从纱帐里面往外看，纱帐是半透明的，外面的人影模模糊糊只能看个大概，像隔着毛玻璃。甜甜和佳佳都以为外面的人看不清——光线大部分来自前面的幕布，背光的情况下纱帐应该是不透的。但纱帐有一个问题——它很薄。当幕布上的画面是明亮的场景时，纱帐上的影子就像投影仪的镜头一样清晰。

佳佳的手伸到甜甜的卫衣下摆。他把她的卫衣从下往上掀起来——纱帐上清晰地映出她的手臂举过头顶的影子。卫衣脱掉了。她的内衣是白色的——蕾丝的。他解她内衣的搭扣，一个动作——两边的肩带从她肩膀上滑落。她的乳房弹出来——纱帐上她的轮廓胸前凸起了两个半圆的弧线。她低头看了一眼自己的胸口——她没有遮。她伸手解了佳佳的裤子拉链。他的裤子滑到脚踝——然后是内裤。

纱帐上——她脱了上衣，他脱了下半身。两个人影的面孔正好朝向窗外的幕布——电影恰好切到了一个大晴天的航拍镜头。光从幕布反射到纱帐上——那一刻纱帐薄到几乎透明。

蒙古包里的炉火烧得噼啪响。珍珍坐在正对纱帐的位置——她能看到的轮廓比任何人都多。她看到佳佳的轮廓把甜甜的轮廓压在窗台上。她看到甜甜的轮廓弯下腰——然后佳佳的轮廓站在她面前，手扶着她的后脑勺。她能从那轮廓的上下移动判断出甜甜的嘴正在含着他的鸡巴。她看到佳佳的轮廓仰起了头——脖子拉成一条直线——那是男人被含到龟头时最自然的反应。珍珍没有移开目光。她能看到甜甜的头前后移动的幅度和频率，能看到佳佳的手从扶着她后脑勺变成插进了她的头发里——他在控制节奏。

珍珍没有移开目光。

阿伟坐在矮桌的另一侧。他也看着纱帐——但他的角度看不到太多细节。他看到佳佳站在窗前，纱帐在他身后合拢。他看不到甜甜的头在佳佳腿间的起伏。但他能从佳佳仰起头的那个动作中猜到一些。

之后没多久纱帐拉开。甜甜走出来——嘴唇比进去时更红。她坐回阿伟旁边的时候阿伟没有说话。


第三轮。屏幕上停住：8号和1号——纱帐，不能用手，只能用嘴让对方硬。

8号是静静。1号是阿贤。

静静站起来的时候她的目光从阿敏脸上掠过——只有一瞬。她走到纱帐前。阿贤跟在她身后。纱帐在两人身后合拢。

电影的画面切换到夜晚的星空——幕布上黑色的夜空上缀满白色的星点。光线很暗——纱帐里几乎完全暗了下来，什么都看不到。

静静背对着窗户。阿贤站在她面前。纱帐里的光线昏暗到只能看到彼此的瞳孔，但在纱帐外面同样什么都看不到。

静静小声说——你躺下。

阿贤躺在了窗台下面的羊毛坐垫上。静静跪下来——膝盖落在垫子上的声音很轻。她的手在他胸口停了一下然后一路往下。她拉开了他的裤链——他的内裤前端已经鼓得老高了。她把他的内裤往下拉——那根鸡巴弹出来差点打到她的下巴。她低头看着它停了一下——又粗又长，龟头已经完全充血了，青筋从根部延伸到冠沟。她用拇指擦了一下龟头——马眼上已经渗出一滴透明的液体。她犹豫了一下，然后低头含住了他。

阿贤的呼吸在那一瞬间变了调——操。

她的嘴很湿，舌头绕着他的龟头打转——动作不快，甚至带着一点生涩。她含得不太深但节奏很好——她的嘴唇每一次退到龟头边缘的时候舌尖都会在龟头与包皮的连接处舔一下，然后重新含进去。她的手握着他的根部配合着嘴的节奏套弄——她的嘴只能含进去一半，因为太长了。外面电影的声音很大——风声和马蹄声盖住了纱帐里大部分的声音。但阿贤能听到她每一次吞咽的细微动静。他伸手把她的头发拢到耳后——她的耳朵烫得厉害。

几分钟后纱帐拉开。静静先走出来，然后阿贤跟在她后面。两个人的表情都很正常——但阿敏注意到静静的嘴唇湿得不太正常。


电影的剧情渐渐没人关心了。到了第四轮的时候，不需要珍珍催——大家已经开始自己期待下一次被叫到的人是哪两个号码。

屏幕上停住：4号和9号——纱帐，脱光对方的所有衣物，但只能用嘴。

4号是阿伟。9号是嘉嘉。

阿伟站起来走进纱帐的时候他看了一眼佳佳——佳佳正靠在坐垫上喝马奶酒，表情看不出任何异样。

纱帐合拢了。电影的画面恰好是一个明亮的草原日景——纱帐上清清楚楚地映出两个人轮廓。阿伟站在嘉嘉面前。嘉嘉背对着窗户站着。她歪了一下头——像在说"你开始吧"。

阿伟弯下腰用嘴咬住了她T恤的下摆往上拉。纱帐上她的手臂举过头顶——T恤从头顶脱掉。她用牙齿解了他衬衫的纽扣——一颗、两颗——她的牙龈碰到纽扣的时候发出的细微声响。她咬着他衬衫的领口往侧面一拽——衬衫从他一边的肩膀滑落。

阿伟蹲下来——嘴唇贴着她短裤的边缘——他用牙齿咬住了她短裤侧面的抽绳结。他的嘴唇和牙齿配合着慢慢拉开那个结——然后咬住短裤的边缘往下拉。她配合地抬起了一只脚——然后又抬起另一只。他的嘴唇贴着她的大腿内侧——在她脱掉短裤的过程中他的嘴唇几乎贴遍了她的整条大腿。

嘉嘉的内裤是一条黑色的丁字裤——两边只有一根细带。阿伟用牙咬住那根细带往下拉的时候她的髋骨两侧留下了一道浅浅的红痕。

现在她全裸了。

她也用嘴脱了他的。等他全裸之后两个人面对面站在纱帐里。纱帐上两具人体的轮廓叠在一起。阿伟伸出手把嘉嘉拉近了一步——纱帐上两个人影的胸口贴在了一起。

电影切到了一个明亮的场景——那一刻纱帐上的画面比任何时候都清晰。嘉嘉的轮廓弯下腰——她的嘴含住了阿伟的轮廓的裆部。阿伟的轮廓仰起了头。嘉嘉的轮廓上下移动的速度很快。

阿敏坐在矮桌旁边——他的位置斜对着纱帐。他拿起酒杯喝了一口然后放下来——他的目光停在纱帐上没有移开过。他旁边坐着小芳。小芳也在看着纱帐——但她看的是阿伟的轮廓。

纱帐里的人对外面的透明度一无所知。嘉嘉含着阿伟的阴茎吞吐。她含得很深，喉咙包裹着龟头，然后退到只剩龟头含在嘴唇之间，再重新吞进去。阿伟的手插在她的头发里——不是控制节奏，只是握着。

电影切到了夜空的画面——纱帐重新变得不透明了。蒙古包里的人只能看到两个模糊的轮廓贴合在一起。但刚才那几秒已经够了。阿敏端起酒杯——他的手有一点稳不住。

纱帐重新拉开的时候阿伟第一个出来。他的裤裆还鼓着。他坐回甜甜旁边的时候甜甜的嘴角动了一下——她说——她口活不错吧。阿伟没有说话。


第五轮。屏幕上停住：10号和6号——纱帐，5分钟，想做什么做什么，不受限制。

10号是小芳。6号是阿敏。

这一轮号码亮出来的时候空气安静了整整三秒。小芳先站起来的——她的动作很平静。她走到纱帐前面回头看了一眼阿敏。阿敏站起来的时候他下意识地看了一眼静静。静静端着奶茶杯在喝——她没有看他。

纱帐合拢了。

电影的画面恰好切到了一段明亮的场景——草原上阳光灿烂。纱帐上映出两个人的轮廓。小芳站在窗前。阿敏站在她面前。

纱帐里小芳低声说——你不用有压力。

阿敏同样的低声回答——这不是有没有压力的问题。

小芳——那是什么问题。

阿敏——是我不确定你是不是真的想在这里。

小芳——我抽到了10号。你抽到了6号。这就是答案。

她伸手解开了自己衬衫的扣子。一颗。两颗。三颗。衬衫从她的肩膀滑落到地上。她的内衣是浅灰色的——棉质的，没有钢圈。她脱了它。她的乳房不大——B罩杯，但乳形很好，乳尖是浅粉色的。她的肤色在从纱帐透进来的光线中白得几乎透明。

阿敏的手抬起来——他的手指碰了一下她的肩膀——像在确认她是不是真的。然后他的手贴着她的锁骨慢慢滑到她的胸口——他的掌心覆住了她的乳房的侧面。她能感受到他掌心的温度——很烫。

小芳低头看着他的手——她说你也脱。

阿敏脱下T恤，然后是裤子。两个人面对面全裸地站在纱帐里。纱帐上他们的轮廓叠在一起。电影恰好放到了一个明亮的草原全景——那一刻纱帐上的画面一清二楚。蒙古包里所有人都能看到——两个全裸的轮廓面对面站着，阿敏的手放在小芳的腰上，小芳的手放在阿敏的胸口。

小芳转过头——她的嘴唇贴着他的脖子，然后一路吻到他的锁骨。他握着她腰的手收紧了一点。他的呼吸声在纱帐的封闭空间里变得清晰可闻。她的手从他的胸口滑到他的小腹——然后握住了他。他已经半硬了。她握着他慢慢套弄——龟头从包皮里露出来的时候他深吸了一口气。

她低下头含住了他。阿敏的腰往前送了一下——动作不大，是他控制不住的反应。她的嘴唇包裹着他的龟头，舌尖在马眼上画了一个圈。他的手指插进了她的头发里——轻轻地，不使劲。

电影正在放草原的黄昏——光线昏暗。纱帐重新变成不透明的了。蒙古包里的人只能看到两个轮廓交叠在一起，看不到具体在做什么。但刚才那几秒已经足够让所有人看到小芳含着他的那一幕。

纱帐里，小芳的口交动作突然停了。她抬起头——她说——外面能看到我们吗。

阿敏——应该看不到吧。纱帐挺厚的。

小芳转头看了一眼蒙古包里的方向——透过纱帐她只能看到模糊的人影在移动，像隔着磨砂玻璃。她确实什么都看不清。

她低头看了一眼自己的胸口——乳尖还湿着，沾着她的唾液在灯光下反光。她伸手握住了他的鸡巴——已经硬得发烫了，龟头从包皮里完全露出来，青筋在她掌心跳动。她上下套弄了两下——阿敏的呼吸跟着她的节奏变粗了。

她犹豫了一下，然后翻身背对着他，双手撑在窗台上——臀部翘起来对着他。她回头看了他一眼——操我。

阿敏走到她身后，手扶着她的胯骨。龟头碰到她阴唇的时候小芳的呼吸断了半拍——她已经湿透了。他的龟头在她阴唇缝隙上下滑动了两下，沾满了她的水，然后对准了入口。

他沉腰——噗呲——那根鸡巴一寸一寸地没入她的骚逼。小芳咬住了自己的手背，眼睛一下子睁大了。他能看到她后颈的肌肉绷紧又放松。他停下来让她适应——她的阴道壁紧紧地裹着他，又热又滑。

她缓了几口气，然后往后顶了一下——别停，操我。

他开始抽送。龟头刮过她阴道壁每一道褶皱——小芳的嘴唇张开了一条缝，压抑的呻吟从牙缝里挤出来——啊——好深——操——好深。她的屁股主动往后顶，迎合他的每一下撞击。她的手在窗台上撑不住滑了一下——胸贴着窗台，乳头蹭过粗糙的木框让她浑身一抖。纱帐上两个人的轮廓合二为一——她的腰弯着，他的身体贴着她的背部。电影的画面在幕布上流动——没有人知道那部纪录片在讲什么。所有人都看着纱帐——即使纱帐现在是暗的，什么都看不到。

但他们的目光还是黏在那里。因为里面的人在做什么——每个人都能从影子的节奏中读出来。


第六轮还没有开始的时候有人发现纱帐的秘密。

静静起身上厕所——她从纱帐旁边经过的时候恰好电影打过来一道强光——她看到纱帐上阿伟和嘉嘉的影子清晰到连手指的动作都能分辨。她停了一下。然后她若无其事地继续走向门口。

阿敏沉默了一会儿——不打算。

静静转头看了他一眼——我也是这么想的。


第七轮。屏幕停住：5号和1号——纱帐，坐上去自己动。

5号是佳佳。1号是阿贤。

两个人站起来的时候佳佳笑了一下——兄弟，没想到吧。阿贤没笑。他跟着佳佳走进了纱帐。

第八轮。屏幕上停住：7号和3号——纱帐，一个跪着，把腿分开，另一个用嘴让她高潮。

纱帐外面——电影的光正好切到明亮处。纱帐上甜甜的轮廓跪在阿贤的轮廓面前——他的胯部的位置对着她的脸。阿贤的轮廓仰着头——脖子拉成一条直线。蒙古包里所有人都看清了那个剪影的每一个动作。


第九轮。屏幕上停住：2号和4号——纱帐，操她，直到她高潮，再换她操你。

2号是甜甜。4号是阿伟。

纱帐上她的轮廓骑着他的轮廓，腰部开始用力起伏。

纱帐里甜甜俯下身——她凑到阿伟耳边——她说——你知道吗，外面可能看得到。

阿伟的身体僵住了一瞬——什么？

甜甜笑——骗你的。纱帐厚着呢。

但她没有说。她反而更用力地骑着他。因为让所有人看到她在骑阿伟——这个想法让她湿透了。


第十轮。屏幕上停住：9号和8号——纱帐，这一次在里面待十分钟。想做什么就做什么。没有限制。

9号是嘉嘉。8号是静静。

两个女生站起来的时候佳佳说了一句——这是要干嘛。嘉嘉回头看了他一眼——她说想知道你就进来一起。

佳佳没说话了。

纱帐合拢。电影在帐篷外面继续放映——画面的光线从幕布上折射进来。纱帐里的人对内幕透明度浑然不觉，但蒙古包里的人不再是最初那种"好奇偷看"的状态了——所有人都坐直了，目光集中在纱帐上。因为他们发现了一件微妙的事：电影的光线强度变成了某种天然的直播信号。

嘉嘉说——我可以吗。

电影突然切到了大晴天的镜头。纱帐近乎全透明地展示了一切——两个人跪坐着的剪影、交叉的手指、身体倾侧的角度、嘴唇落在胸口上的那个瞬间。蒙古包里没有人作声。

纱帐里，静静突然凑到嘉嘉耳边说——外面是不是能看到。

嘉嘉没有否认。她低声说——能看到——很清晰。你的大奶子很白——外面的人应该也看到了。

静静的身体僵了一下——但她没有退开。她沉默了几秒后说——你从什么时候知道的。

嘉嘉——第一轮就知道了。阿强和我的时候我发现电影亮的时候纱帐是透的。

静静——那你为什么不告诉我。

嘉嘉——因为你现在还没退开。

第十一轮。屏幕上停住：6号和10号——没有纱帐。就在这里。

所有人。

不是两个人。是所有人一起。

所有人都站了起来。纱帐被彻底拉开。窗外的电影还在放——幕布上光影流转。蒙古包内炉火正旺。

蒙古包里不再有明区和暗区的区别。

纱帐被拉开了。

阿贤第一个动。他走到小芳面前——看了阿敏一眼。阿敏没有动——他坐在坐垫上看着这一切。阿贤在小芳面前蹲下来——他伸手掰开她的腿——她没反抗。他的嘴埋进了她的腿间——她的逼在纱帐里被阿敏操过之后还红肿着，阴唇外翻，逼口还在往外渗透明的液体。阿贤的舌头舔了上去——小芳发出一声短促的抽气。阿敏看着他舔舐自己女朋友的逼——嘴角动了一下——但他没有阻止。

静静趴在羊毛坐垫上——阿伟从后面操她。他的鸡巴在她逼里进进出出——带出白色的泡沫。静静地咬着垫子边缘——电流一样的快感从尾椎一路窜到后脑勺。甜甜跪在她面前——掰开她的嘴往她嘴里塞了一根不知道是谁的鸡巴——吃——你今天就该吃这根鸡巴。静静听话地含了进去——她甚至没有看是谁的。龟头顶到她喉咙深处的时候她的眼睛翻了一下白。

阿强从嘉嘉腿间抬起头——他嘴里全是她的水。他翻身把嘉嘉压在身子下面——龟头蹭着她湿淋淋的逼口然后一挺腰操了进去。嘉嘉的腿缠上了他的腰——操我——用力操我——你的大鸡巴——她的话被撞成了断断续续的气音。

电影的光在某个瞬间亮了起来——那道光扫过蒙古包内部闪过了所有人的剪影。阿伟跪在静静身后操她——她的屁股悬在半空中。珍珍骑在阿贤身上——她的腰在起伏——她的大奶子跟着节奏跳动。佳佳跪在珍珍身后——他的鸡巴在她嘴里进进出出。阿强和嘉嘉在地面上交缠在一起——她的腿架在他肩膀上。甜甜趴在矮桌上——阿敏的手指在她逼里——她自己的手指在她自己的阴蒂上揉搓。

光暗了下去——剪影重新没入黑暗中。

凌晨时分，电影放完了。幕布上变成了白色的光板。纱帐垂在窗边，恢复了作为装饰品的安静姿态。蒙古包里横七竖八地躺着人。有人裹着毯子，有人垫着衣服，有人就仰面躺着看着包顶发呆。

炉火烧成了暗红色的炭。阿贤靠着包壁坐着。小芳的头枕在他大腿上——她闭着眼睛但没睡着。

阿贤低头看了她一眼。她的睫毛在火光中微微颤动。

他说——你今天在纱帐里和阿敏的时候电影正亮。

小芳的睫毛没有动——我知道。

她沉默了一会儿——我看到甜甜骑你的时候电影也亮着。

阿贤——你知道。

小芳——嗯。

她睁开眼睛——我也没走。

她重新闭上眼睛。炉火噼啪响了一声。草原的夜晚很安静。月亮从云层后面移出来——月光透过纱帐在地板上画出一道一道白色的光栅。
